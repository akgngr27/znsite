Stack Kütüphanesi

Stack kütüphanesi Strings kütüphanesinin yöntemlerinden üretilmiş yeni bir kütüphanedir. Strings kütüphanesinde yer alan yöntemleri çoklu olarak kullanabilmeyi sağlar.

İşlem Görecek Dizgeyi Belirlemek

Stack::data( string $dizi )

İşlem görecek dizi belirtilir. Bu yöntem Stack:: dan hemen sonra belirtilmelidir yani en üst fonksiyon olarak kullanılmalıdır.

NoParametrelerAnlamları
1string DiziDeğişikliğin hangi dizide yapılacağı.

$dizge = 'test';
Stack::data($dizge )
		  -> ...
          -> ...
          ->get();

İşlem Sonucunu Almak

Stack::get()

İşlem sonucunu almak için kullanılır. Bu yöntem nihai(son) yöntem olarak kulanılmalıdır.

$dizge = 'test';
Stack::data($dizge )
		  ->uppercase()  // Bu arada Strings kütüphanesine ait herhangi bir fonskiyon veya fonksiyonlar kullanılabilir. 
          ->length()
          -> ...
          ->get();

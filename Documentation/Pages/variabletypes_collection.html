Collection Kütüphanesi

Collection kütüphanesi Arrays kütüphanesinin yöntemlerinden üretilmiş yeni bir kütüphanedir. Arrays kütüphanesinde yer alan yöntemleri çoklu olarak kullanabilmeyi sağlar.

İşlem Görecek Diziyi Belirlemek

Collection::data( array $dizi )

İşlem görecek dizi belirtilir. Bu yöntem Collection:: dan hemen sonra belirtilmelidir yani en üst fonksiyon olarak kullanılmalıdır.

NoParametrelerAnlamları
1array DiziDeğişikliğin hangi dizide yapılacağı.

$dizi = array(1, 2, 3);
Collection::data($dizi )
		  -> ...
          -> ...
          ->get();

İşlem Sonucunu Almak

Collection::get()

İşlem sonucunu almak için kullanılır. Bu yöntem nihai(son) yöntem olarak kulanılmalıdır.

$dizi = array(1, 2, 3);
Collection::data($dizi )
		  ->removeFirst()  // Bu arada Arrays kütüphanesine ait herhangi bir fonskiyon veya fonksiyonlar kullanılabilir. 
          ->removeLast()
          -> ...
          ->get();

HTML5 Form Input Nesneleri

Html::input( string $type , string $isim , string $deger , [ array $ozellikler ] )

HTML5 tüm giriş nesnelerinin 3 parametresi vardır. Nesne İsmi, Nesne Değeri, Nesne Özellikleri

ParametreAnlamları
1. Parametre = Nesne TürüHTML5 nesnesinin türü. email, image, url, date
2. Parametre = Nesne İsmiHTML5 nesnesinin ismi.
3. Parametre = Nesne DeğeriHTML5 nesnesinin üzerinde görünecek yazı.
4. Parametre = [Nesne Özellikleri]HTML5 nesnesinin özellik ve değerleri.

E-posta Nesnesinin Kullanımı

echo Html::input('email', 'email', 'Lütfen e-posta adresinizi giriniz.', array('maxlength' => '10'));
// <input type="email" name="email" id="email" value="Lütfen e-posta adresinizi giriniz." maxlength="10">

URL Nesnesinin Kullanımı

echo Html::input('url', 'urlAdresi', 'Lütfen URL adresinizi giriniz.', array('maxlength' => '10'));
// <input type="url" name="urlAdresi" id="urlAdresi" value="Lütfen URL adresinizi giriniz." maxlength="10">

NOT
Diğer HTML5 form nesneleride aynı formatta kullanılmaktadır.

HTML5 İçerik Düzenleme Nesneleri

Html::xxx( string $icerik , [ array $ozellikler ] )

HTML5 tüm içerik düzenleme nesnelerinin 3 parametresi vardır. Nesne İsmi, Nesne Değeri, Nesne Özellikleri

ParametreAnlamları
1. Parametre = İçerikHTML5 nesnesinin içeriği.
2. Parametre = [Nesne Özellikleri]HTML5 nesnesinin özellik ve değerleri.

Canvas Nesnesinin Kullanımı

echo Html::header('Üst Bilgi', array('id' => 'ustBilgi'));
// <header id="ustBilgi">Üst Bilgi</header>

Canvas Nesnesinin Kullanımı

echo Html::summary('Özet Bilgi', array('id' => 'ozet'));
// <summary id="ozet">Özet Bilgi</summary>

NOT
Diğer HTML5 içerik düzenleme nesneleride aynı formatta kullanılmaktadır.

HTML5 Multimedya Nesneleri

Sesler ve görüntülerden yararlanmak için kullanılan nesnelerdir.

Ses Nesnesi

Html::auido( string $kaynak , string $icerik , [ array $ozellikler ] )

ParametreAnlamları
1. Parametre = KaynakSes dosyasının çekildiği kaynak.
2. Parametre = İçerikAudio nesnesinin içeriği.
3. Parametre = [Nesne Özellikleri]Audio nesnesinin özellik ve değerleri.

echo Html::audio('http://www.ornek.com/ses.mp4', 'Ses Dosyası', array('id' => 'sesId'));
// <audio src="http://www.ornek.com/ses.mp4" id="sesId">Ses Dosyası</audio>

Görüntü Nesnesi

Html::video( string $kaynak , string $icerik , [ array $ozellikler ] )

ParametreAnlamları
1. Parametre = KaynakGörüntü dosyasının çekildiği kaynak.
2. Parametre = İçerikVideo nesnesinin içeriği.
3. Parametre = [Nesne Özellikleri]Video nesnesinin özellik ve değerleri.

echo Html::video('http://www.ornek.com/film.mpg', 'Görüntü Dosyası', array('id' => 'goruntuId'));
// <video src="http://www.ornek.com/film.mpg" id="sesId">Görüntü Dosyası</video>

echo Html::source('http://www.ornek.com/film.mpg', array('id' => 'goruntuId'));
// <source src="http://www.ornek.com/film.mpg" id="goruntuId" />

Güvenlik Kodu Kütüphanesi Yöntemleri

Güvenlik Kodu kütüphanesine ait yöntemler aşağıdaki tabloda listelenmiştir. Kod üzerinden güvenlik kodu ayarlarını yapılandırabileceğiniz gibi Config/Components.php ayar dosyasından da yapılandırmaları gerçekleştirebilirsiniz.

Güvenlik Kodu Oluşturma YöntemleriAnlamları
width($param = 0) Oluşturulacak güvenlik kodunun genişlik değeridir. Bu yöntem kullanılmadığı taktirde varsayılan genişlik değerini Config/Captcha.php dosyasındaki genişlik değerine göre ayarlayacaktır.
height($param = 0) Oluşturulacak güvenlik kodunun yükseklik değeridir. Bu yöntem kullanılmadığı taktirde varsayılan yükseklik değerini Config/Captcha.php dosyasındaki yükseklik değerine göre ayarlayacaktır.
size($width = 0, $height = 0) Width() ve Height() yönteminin alternatifi olarak oluşturulmuştur. Oluşturulacak güvenlik kodunun genişik ve yükseklik değerini ayarlamak için kullanılır. Bu yöntem kullanılmadığı taktirde varsayılan genişik yükseklik değerini Config/Captcha.php dosyasındaki genişlik ve yükseklik değerine göre ayarlayacaktır.
length($param = 0) Güvenlik kodunda yer alan metnin kaç karakterden oluşacağını ayarlamak için kullanılır. Bu yöntem kullanılmadığı taktirde varsayılan karakter uzunluğu(char_count) değerini Config/Captcha.php dosyasındaki karakter uzunluğu değerine göre ayarlayacaktır.
border($is = false, $color = '') Güvenlik kodu resminde çerçeve olup olmayacağını ayarlamak için kullanılırken ayrı zamanda bu çerçevenin rengide ayarlanabilir.
1. parametre çerçevenin olup olmayacağını ayarlar ve true veya false değeri alır.
2. parametre ise renginin ne olacağını belirtilir. Parametre olarak RGB standartına uygun metinsel değer girilir. Örnek: 100|255|190.
Bu yöntem kullanılmadığı taktirde varsayılan çerçeve değerlerini Config/Captcha.php dosyasındaki çerçeve değerlerine göre ayarlayacaktır.
borderColor($color = '') Border() yönteminin 2. parametresi ile aynı işi görür. Yani çerçevenin rengini ayarlamak için kullanılır. Parametrenin kullanımı yukarıda anlatıldığı gibidir.
Bu yöntem kullanılmadığı taktirde varsayılan çerçeve rengi değerini Config/Captcha.php dosyasındaki çerçeve rengi değerine göre ayarlayacaktır.
bgColor($color = '') Güvenlik kodunun arkaplan rengini ayarlamak için kullanılır. Parametrenin kullanımı yukarıda anlatıldığı gibidir.
Bu yöntem kullanılmadığı taktirde varsayılan arkplan rengi değerini Config/Captcha.php dosyasındaki arkaplan rengi değerine göre ayarlayacaktır.
bgImage($image = array()) Güvenlik kodunun arkaplanında yer alacak resimleri ayarlamak için kullanılır. Arkaplan için tek bir resim kullanılacaksa string türde veri girilebilir. Ancak Birden fazla resim bilgisi eklenecekse dizi türünde veri girilmelidir.
background($background = '') Bgimage() ve Bgcolor() yöntemlerine alternatif olarak oluşturulmuştur. İster metinsel olarak R|G|B formunda renk kodu girebilirsiniz. İsterseniz de dizi türünde arkaplan resimleri ekleyebilirsiniz. Bu yöntem yukarıda anlatılan 2 yöntemin işinide üstlenir.
textSize($size = 0) Güvenlik kodu metninin boyutunu ayarlamak için kullanılır. Bu yöntem kullanılmadığı takdirde varsayılan ayar kullanılacaktır.
textCoordinate($x = 0, $y = 0) Güvenlik kodu metninin x ve y değerlerini ayarlamak için kullanılır. Bu yöntem kullanılmadığı takdirde varsayılan ayar kullanılacaktır.
textColor($color = '') Güvenlik kodu metninin rengini ayarlamak için kullanılır. R|G|B standartına uygun olarak metinsel veril girilmelidir. Bu yöntem kullanılmadığı takdirde varsayılan ayar kullanılacaktır.
text($size = 0, $x = 0, $y = 0, $color = '') text_size(), text_coodinate() ve text_color() yöntemlerinin alternatifi olarak kullanılabilir. Belirtilen bu yöntemlerin işini tek seferde yapabilir. Bu yöntem kullanılmadığı takdirde varsayılan ayarlar kullanılacaktır.
grid($is = false, $color = '') Güvenlik kodu resminde ızgara olup olmayacağını ayarlamak için kullanılırken ayrı zamanda bu ızgaranın rengide ayarlanabilir.
1. parametre ızgaranın olup olmayacağını ayarlar ve true veya false değeri alır.
2. parametre ise renginin ne olacağını belirtilir. Parametre olarak RGB standartına uygun metinsel değer girilir. Örnek: 100|255|190.
Bu yöntem kullanılmadığı taktirde varsayılan çerçeve değerlerini Config/Captcha.php dosyasındaki ızgara değerlerine göre ayarlayacaktır.
gridSpace($x = 0, $y = 0) Izgaraların yatayda ve dikeyde kaçar adet olacağını belirlemek için kullanılır.
1. parametre: yatayda kaç çizgi olacağını belirler.
2. parametre: dikeyde kaç çizgi olacağını belirler.
Bu yöntem kullanılmadığı taktirde varsayılan ızgara değerini Config/Captcha.php dosyasındaki ızgara değerine göre ayarlayacaktır.
gridColor($color = '') Grid() yönteminin 2. parametresi ile aynı işi görür. Yani ızgara rengini ayarlamak için kullanılır. Parametrenin kullanımı yukarıda anlatıldığı gibidir.
Bu yöntem kullanılmadığı taktirde varsayılan ızgara rengi değerini Config/Captcha.php dosyasındaki ızgara rengi değerine göre ayarlayacaktır.
create($img = false) Güvenlik kodu oluşturma yöntemidir. En son bu yöntem kullanılarak güvenlik kodu oluşturulması sağlanır. 1 parametresi vardır. Bu parametre true olarak ayarlandığı zaman güvenlik kodu linkini html img etiketi içerisinde kullanarak çıktılar.

Yukarıdaki yöntemlerin kullanımına yönelik örnekler aşağıda verilmiştir.

// Metnin kaç karakter olacağı ayarlanıyor.
// Kullanımı zorunlu değildir. Varsayılan: 6
Captcha::length(6) 
       // Metnin rengi ayarlanıyor.
       // Kullanımı zorunlu değildir. Varsayılan: 255|255|255
       ->textColor('180|20|10') 
       // Grid ve rengi ayarlanıyor.
       // Kullanımı zorunlu değildir. Varsayılan: true, 50|50|50
       ->grid(true, '30|30|30') 
       // Yatayda ve dikeyte ızgara sayısı ayarlanıyor.
       // Kullanımı zorunlu değildir. Varsayılan: 12, 4
       ->gridSpace(2, 6) 
       // Güvenlik kodunun arkaplan rengi ayarlanıyor.
       // Kullanımı zorunlu değildir. Varsayılan: 80|80|80
       ->background('200|200|200') 
       // Güvenlik kodunu oluşturan nihai yöntemdir.
       ->create(true);

Yukarıda kodların çıktısı olan güvenlik kodu görüntüsü yer almaktadır.

Güvenlik Kodunu Öğrenmek

Captcha::getCode()

Oluşturulan güvenlik kodunu öğrenmek için kullanılır.

echo Captcha::getCode(); // Çıktı: 1A4D31

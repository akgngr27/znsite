Buffer(Tamponlama) Kütüphanesi

Tampon kullanımı için oluşturulmuş sınıftır.

Buffer(Tamponlama) Kütüphanesi ve Yöntemleri
Dosya İçeriğini TamponlamakBuffer::file()
Yöntem Çıktılarını TamponlamakBuffer::func()
Tampona Veri EklemekBuffer::insert()
Tampondan Veri SeçmekBuffer::select()
Tampondan Veri SilmekBuffer::delete()

Dosya İçeriğini Tamponlamak

Buffer::file( string $dosyaAdi )

Dosyaları tampona almak için kullanılır.

NoParametrelerAnlamları
1string Dosya YoluTampona alınacak dosya yolu.

$data = Buffer::file('Dosya/test.php');
echo $data;

Yöntem Çıktılarını Tamponlamak

Buffer::func( string $yontemAdi , [ array $parametreler ] )

Yöntemlerin çıktılarını tampona almak için kullanılır.

NoParametrelerAnlamları
1string Yöntem AdıTampona alınacak yöntemin adı.
2[array Yöntem Parametreleri]Varsa yöntemin parametreleri.

$data = Buffer::func(function(){ return 1; });
echo $data; // Çıktı: 1

$data = Buffer::func('get_defined_constants', array(true));
output($data);

$data = Buffer::func('get_defined_functions');
output($data);

Tampona Veri Eklemek

Buffer::insert( string $veriAdi , mixed $deger , [ array $params ] )

Tampona veri eklemek için kullanılır.

ParametreAnlamları
1. Parametre = string Veri AdıOluşturulacak verinin adı.
2. Parametre = mixed DeğerOluşturulacak verinin değeri.
[3. Parametre = array Parametreler]Tampona alınacak veri fonkisyon ise o fonksiyona ait parametreler.

Buffer::insert('veri1', 'Değer');

Buffer::insert('veri2', function(){ return 1; });

Buffer::insert('veri3', 'get_defined_constants', array(true));

Buffer::insert('veri4', 'Dosya/index.html');

Tampondan Veri Seçmek

Buffer::select( string $veriAdi )

Seçilecek verinin adı.

NoParametrelerAnlamları
1string Veri AdıTampona alınacak verinin adı.

echo Buffer::select('veri1');

Tampondan Veri Silmek

Buffer::delete( string $veriAdi )

Silinecek verinin adı.

NoParametrelerAnlamları
1string Veri AdıTampondan silinecek verinin adı.

echo Buffer::delete('veri1');

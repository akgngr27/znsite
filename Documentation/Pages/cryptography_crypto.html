Crypto(Şifreleme) Kütüphanesi

Şifreleme işlevlerini yerine getiren sınıftır. Şifrelemelerde kullanılan vektör ve anahtarlar için varsayılan olarak Config/Encode.php dosyasında yer alan projectKey parametresindeki değerden yararlınılmıştır. Kendi vektör ve anahtar değerlerinizi belirtmeniz durumunda kişisel ayarlamalarınız geçerli olacaktır.

Veriyi Şifrelemek

Crypto::encrypt( string $veri , array $parametreler );

Veriyi şifrelemek için kullanılır.

ParametrelerAnlamları
Destekleyen Sürücülermcrypt, openssl, hash, mhash
1 - string Şifrelenecek VeriÇıkarma işlemin yapılacağı sıkıştırılmış dosya.
2 - [array Parametreler]Şifrelemede kullanacak diğer ayarlar.
2. Parametrede Kullanılabilir Anahtarlar
cipherŞifreleme türü. Varsayılan mcrypt:des, openssl:aes-128, hash:sha256, mhash:sha256.
keyŞifreleme anahtarı. Varsayılan Tüm sürücüler için Config/Encode.php'de yer alan projectKey değeridir.
modeŞifreleme modu. Varsayılan mcrypt::cbc, openssl:cbc
vectorŞifrelede kullanılan IV(Vector) vektör parametresi. Varsayılan uzunluklar.

echo Crypto::encrypt('Şifreli Veri')

Farklı Sürücüler Kullanmak

Crypto::driver( string $surucuIsmi )->...

Farklı bir sürücü kullanmak için bu yönteme parametre olarak geçerli şifreleme sürücülerinden birinin ismi yazılır.

ParametrelerAnlamları
1 - string Sürücü İsmiKullanılabilir sürücüler.

echo Crypto::driver('mcrypt')->encrypt('Şifre');
echo Crypto::driver('openssl')->encrypt('Şifre');

Şifreli Veriyi Çözmek

Crypto::decrypt( string $veri , array $parametreler )

Veriyi şifrelemek için kullanılır.

ParametrelerAnlamları
Destekleyen Sürücülermcrypt, openssl
1 - string Şifrelenecek VeriÇıkarma işlemin yapılacağı sıkıştırılmış dosya.
2 - [array Parametreler]Şifrelemede kullanacak diğer ayarlar.
2. Parametrede Kullanılabilir Anahtarlar
cipherŞifreleme türü. Varsayılan mcrypt:des, openssl:aes-128
keyŞifreleme anahtarı. Varsayılan Tüm sürücüler için Config/Encode.php'de yer alan projectKey değeridir.
modeŞifreleme modu. Varsayılan mcrypt::cbc, openssl:cbc
vectorŞifrelede kullanılan IV(Vector) vektör parametresi. Varsayılan uzunluklar.

echo Crypto::decrypt( Crypto::encrypt('Şifreli Veri') );

İstenilen Uzunlukta Şifre Üretmek

Crypto::keygen( numeric $uzunluk )

Belirli uzunlukta şifre üretmek için kullanılır.

ParametrelerAnlamları
Destekleyen Sürücülermcrypt, openssl, hash, mhash, phash
1 - numeric UzunlukOluşturulmak istenen ragele şifrenin uzunluğu.

echo Crypto::keygen(10);

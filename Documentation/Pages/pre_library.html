Hızlı Kütüphane ve Yöntemlerini Kullanmak

library( string/array $sinif_dosyasi_adi , string $fonksiyon_adi , [ array $parametreler ] )

Bu yöntem kullanmak istediğiniz kütüphanenin yöntemlerine kolay erişim sağlamak için oluşturulmuştur.

NoParametrelerAnlamları
1string/array Sınıf AdıÇağrılacak kütüphanesin sınıf yada dosya adı.
2string FonksiyonKütüphanenin kullanılacak olan yönteminin adı.
3[ array Parametreler ]Varsa yöntemin parametreleri.

echo library('Encode', 'golden', array('Metni Şifrele', 'Şifre Eki')); // Çıktı: bc35185a25c5aa7330114a28525bee76
library('Cookie', 'insert', array('yeni_cerez', 'Yeni Değer'));
echo library('Cookie', 'select', array('yeni_cerez')); // Çıktı: Yeni Değer

Doğrudan Kütüphane Kullanmak

uselib( string $sinif_adi ) v1.4

Bu yöntem, bir kütüphaneyi doğrudan kullanmak için kullanılır. Her kullanımında yeni bir New kullanımı gerektirdiği için tek sefer kullanımı uygundur.

NoParametrelerAnlamları
1string Sınıf AdıÇağrılacak kütüphanesin sınıf yada dosya adı.

print( uselib('DB')->get('Download')->result() );

Sadece kütüphane değil aynı zamanda bileşenlerde kullanılabilir.

print( uselib('CCss')->file('style')->getSelector('bold') );

Bir değişkene atama ile kullanım.

$db = uselib('DB');
print( $db->get('Download')->result() );

Bir Dizi Elemanının Yerini Değiştirmek

Arrays::posChange( array $dizi , string/int $eskiPozisyon , string/int $yeniPozisyon )

Dizide istenilen elemanın pozisyonunu değiştirmek için kullanılır.

NoParametrelerAnlamları
1array DiziDeğişikliğin hangi dizide yapılacağı.
2string/int Eski PozisyonDeğişiklik yapılacak elemanın pozisyon numarası.
3string/int Yeni PozisyonYeni pozisyon numarası.

$dizi = array("a", "b", "c", "d", "e");		
var_dump(Arrays::posChange($dizi, 0, 4));  // Çıktı: b c d e a 
var_dump(Arrays::posChange($dizi, 2, 3));  // Çıktı: a b d c e

Dizideki eleman isimlerini kullanarak yer değiştirmek.

$dizi = array("a", "b", "c", "d", "e");		
var_dump(Arrays::posChange($dizi, "a", "c"));  // Çıktı: b c a d e 
var_dump(Arrays::posChange($dizi, "b", "e"));  // Çıktı: a c d e b

Dizideki eleman numarası ve ismini kullanarak yer değiştirmek.

$dizi = array("a", "b", "c", "d", "e");		
var_dump(Arrays::posChange($dizi, 1, "c"));  // Çıktı: a c b d e 
var_dump(Arrays::posChange($dizi, "b", 4));  // Çıktı: a c d e b

Bir Dizi Elemanının Yerini Değiştirmek

Arrays::posReverse( array $dizi , mixed $eskiPozisyon, mixed $yeniPozisyon )

Dizide istenilen iki elemanın birbiri ile yer değiştirmesi için kullanılır.

NoParametrelerAnlamları
1array DiziDeğişikliğin hangi dizide yapılacağı.
2string/int Yer Değiştirecek ElemanYer değişikliği yapılacak dizi elemanı.
3string/int Yer Değiştirecek ElemanYer değişikliği yapılacak dizi elemanı.

$dizi = array("a", "b", "c", "d", "e");		
var_dump(Arrays::posReverse($dizi, 0, 4));  // Çıktı: e b c d a 
var_dump(Arrays::posReverse($dizi, 1, 3));  // Çıktı: a d c b e

Dizideki eleman isimlerini kullanarak yer değiştirmek.

$dizi = array("a", "b", "c", "d", "e");		
var_dump(Arrays::posReverse($dizi, "a", "c"));  // Çıktı: c b a d e 
var_dump(Arrays::posReverse($dizi, "b", "e"));  // Çıktı: a e c d b

Dizideki eleman numarası ve ismini kullanarak yer değiştirmek.

$dizi = array("a", "b", "c", "d", "e");		
var_dump(Arrays::posReverse($dizi, 1, "c"));  // Çıktı: a c b d e 
var_dump(Arrays::posReverse($dizi, "b", 4));  // Çıktı: a e c d b

Diziden Eleman Silmek

Arrays::deleteElement( array $dizi , mixed $eleman )

Dizide istenilen elemanı silmek için kullanılır.

NoParametrelerAnlamları
1array DiziDeğişikliğin hangi dizide yapılacağı.
2array/string/int Silinecek ElemanSilinecek dizi elemanı veya elemanları.

$dizi = array("a", "b", "c", "d", "e");		
var_dump(Arrays::deleteElement($dizi, 1));  // Çıktı: a c d e  
var_dump(Arrays::deleteElement($dizi, 3));  // Çıktı: a b c e

Dizideki eleman isimlerini kullanarak silmek.

$dizi = array("a", "b", "c", "d", "e");		
var_dump(Arrays::deleteElement($dizi, "a"));  // Çıktı: b c d e
var_dump(Arrays::deleteElement($dizi, "c"));  // Çıktı: a b d e

Dizideki elemanları toplu olarak silmek.

$dizi = array("a", "b", "c", "d", "e");		
var_dump(Arrays::deleteElement($dizi, array("a", "c")));  // Çıktı: b d e 
var_dump(Arrays::deleteElement($dizi, array(1, 2)));  // Çıktı: a d e

NOT
Anahtar değer çifti içeren dizilerde ister anahtar ifadeyi isterseniz de değeri kullanarak verileri silebilirsiniz.

Dizinin Anahtar veya Değer Bilgisine Ulaşmak

Arrays::keyval( array $dizi , [ string $tip = 'val' ] )

Dizinin anahtarına veya değerine ulaşmak.

NoParametrelerAnlamları
1array DiziAnahtarın yada değerin istendiği dizi.
2[ string Tip = "val" ]Anahtar mı değer mi.
2. Parametrenin Alabileceği DeğerlerAnlamları
val veya valueDizinin değerini metinsel veri tipi olarak döndürür.
keyDizinin anahtarını metinsel veri tipi olarak döndürür.
vals veya valuesDizinin değer veya değerlerini dizi veri tipi olarak döndürür.
keysDizinin anahtar veya anahtarlarını dizi veri tipi olarak döndürür.

$dizi = array("a" => "b");		
print(Arrays::keyval($dizi));  // Çıktı: b  
print(Arrays::keyval($dizi, "key"));  // Çıktı: a  
$dizi = array("1" => "a", "2" => "b");		
var_dump(Arrays::keyval($dizi, "values")));  // Çıktı: a, b  
var_dump(Arrays::keyval($dizi, "keys"));  // Çıktı: 1, 2

Arrays::objectData( array $dizi )

NoParametrelerAnlamları

echo Arrays::objectData(array(1 => true, 2 => false)); // {1:true, 2:false}

Dizilerde Sıralama Yapmak

Arrays::order( array $dizi, string $siralamaTuru, string $siralamaKipi = 'regular')

Dizilerde belli kriterlere göre sıralama yapmak için kullanılır. 12 Farklı dizi sıralama yönemi bu fonksiyon altında toplanmıştır.

NoParametrelerAnlamları
1array DiziSıralama yapılacak dizi.
string Sıralama TürüSıralama türü.
ParametrelerAnlamları
ascBir dizinin değerlerini anahtarlarıyla ilişkilerini bozmadan sıralar.
descBir dizinin değerlerini anahtarlarıyla ilişkilerini bozmadan tersine sıralar.
asckeyBir diziyi anahtarlarına göre sıralar.
desckeyBir diziyi anahtarlarına göre tersine sıralar.
userBir diziyi kullanıcı tanımlı bir işlev kullanarak değerlerine göre sıralar.
userassocBir diziyi kullanıcı tanımlı bir işlev kullanarak indislerine göre sıralarken anahtar/değer ilişkisini korur.
userkeyBir diziyi kullanıcı tanımlı bir işlev kullanarak anahtarlarına göre sıralar.
insensBir diziyi harf büyüklüğüne duyarsız "doğal sıralama" algoritmasıyla sıralar.
natural Bir diziyi "doğal sıralama" algoritmasıyla sıralar.
reverseBir diziyi tersine sıralar.
randomBir diziyi rasgele sıralar.
defaultBir diziyi sıralar.
string Sıralama Kipi = RegularSıralama kipi.
ParametrelerAnlamları
regularElemanlar normal olarak sıralanır.
numericElemanlar sayısal olarak sıralanır.
stringElemanlar dizgesel olarak sıralanır.
locale_stringYerele göre dizgesel sıralama yapılır.
flag_caseKip anahtarına göre sıralama yapılır.
naturalDoğar sıralama yapılır.

$dizi = array('1', '2', '3');

output( Arrays::order($dizi, 'random') );
output( Arrays::order($dizi, 'asc', 'natural') );
output( Arrays::order($dizi, 'desc') );

Config/Record.php Dosyasından Kayıt İsmi Belirtmek

Bu ayar dosyasında yer alan record parametresine ön tanımlı olarak kullanılacak kayıt adı belirtilebilir. Veritabanı ismi belirtmek gibi düşünülebilir.

Config/Record.php 
$config['Record']['record'] = 'Test';  // Ön tanımlı olarak "Test" isimlendirmesi kullanılmıştır.

Kayıt Oluşturmak

Record::createRecord( string $record )

Kayıtların tutulacağı dizini oluşturmak için kullanılır. Yukarıdaki ayar kullanılmayacaksa bu yöntemle bir kayıt oluşturulmalıdır.

ParametreAnlamları
1. Parametre = RecordKayıtların tutulması için oluşturulacak dizinin adı.

Record::createRecord('Test');  // Storage/ZNRecords/Test adlı dizin oluşturuldu.

Kayıt Seçmek

Record::selectRecord( string $record )

Oluşturulan kayıt dizinin seçmek için kullanılır.

ParametreAnlamları
1. Parametre = RecordSeçilecek kayıt dizini.

Record::selectRecord('Test');  // Test dizini seçildi.

Kayıta Tablo Oluşturmak

Record::createTable( string $table )

Oluşturulan kayıta tablo yani tablo verilerini tutacak dosyayı oluşturmak için kullanılır.

ParametreAnlamları
1. Parametre = Tablo AdıKayıt içine oluşturulacak dosyanın adı.

Record::createTable('uyeler');  // ../Test/uyeler.znr.php adlı dosya oluşturuldu.

Tablo Seçmek

Record::table( string $table )

Oluşturulan dosyalardan birini seçmek için kullanılır.

ParametreAnlamları
1. Parametre = Tablo AdıSeçilecek dosya adı.

Record::table('uyeler');  // Yukarıda oluşturulan uyeler.znr.php dosyası seçildi.

Tabloya Veri Eklemek

Record::insert( string $table , array $data )

Tabloya eklenecek veriler.

ParametreAnlamları
1. Parametre = Tablo AdıVerilerin hangi dosyaya ekleneceği.
2. Parametre = VerilerTabloya hangi verilerin ekleneceği.

Record::selectRecord('Test');  // Config/Record.php ayar dosyasından ön tanımlı ayarlanmışsa bu komutu kullanmanıza gerek yoktur.
Record::insert('uyeler', array('isim' => 'zntr', 'numara' => 100));  // uyeler.znr.php dosyasına belirtilen veriler eklendi.

table() yönetmini kullanarak da tablo ismi belirtilebilir.

Record::selectRecord('Test');  // Config/Record.php ayar dosyasından ön tanımlı ayarlanmışsa bu komutu kullanmanıza gerek yoktur.
Record::table('uyeler')->insert(array('isim' => 'zntr', 'numara' => 100));  // insert() yönteminin 1. parametresi artık 2. parametre olan data parametresi olduğuna dikkat ediniz.

Tablodan Verileri Seçmek

Record::select( string $table , [ numeric $whereId ] )

Record::get( string $table , [ numeric $whereId ] )

Tablodan veri seçmek için kullanılırlar.

ParametreAnlamları
1. Parametre = Tablo AdıTablo adı.
[2. Parametre = Kayıt Numarası]İstenilen kayıtın numarası.

Record::select('uyeler')->result();  // Tüm veriler object veri türünde elde edildi.
Record::select('uyeler')->resultArray();  // Tüm veriler dizi veri türünde elde edildi.
Record::select('uyeler')->row();  // Tek bir kayıt elde edildi.
    
Record::select('uyeler', 5)->result();  // 5 numaralı veri elde edildi.
NOT
Kayıtların numaraları 1'den başlamaktadır.

Row() yöntemine parametre belirtilerek ilk, son veya istediğiniz kayıtı seçebilirsiniz.

Record::select('uyeler')->row('last');  // Son veri elde edildi.
Record::select('uyeler')->row();  // İlk veri elde edildi.
Record::select('uyeler')->row(3);  // 3 nnumaralı veri elde edildi.
    
Record::select('uyeler', 5)->result();  // 5 numaralı veri elde edildi.

Aşağıdaki gibi bir kullanımda mümkündür.

Record::table('uyeler')->where(5)->get()->row();  // 5 numaralı veri elde edildi.

Tablodaki Veriyi Güncellemek

Record::update( string $table , array $newData, [ numeric $whereId ] )

Tablodan veri seçmek için kullanılırlar.

ParametreAnlamları
1. Parametre = Tablo AdıTablo adı.
2. Parametre = Yeni Kayıt DizisiYeni değerlerin yazılacağı dizi.
[3. Parametre = Kayıt Numarası]İstenilen kayıtın numarası.

Record::update('uyeler', array('isim' => 'ZNTR', 'numara' => 200), 5);  // 5 Numaralı veriyi güncelle.

Ya da

Record::where(5)->update('uyeler', array('isim' => 'ZNTR', 'numara' => 200));  // 5 Numaralı veriyi güncelle.

Ya da

Record::table('uyeler')->where(5)->update(array('isim' => 'ZNTR', 'numara' => 200));  // 5 Numaralı veriyi güncelle.

Tablodaki Veriyi Silmek

Record::delete( string $table , [ numeric $whereId ] )

Tablodan veriyi silmek için kullanılırlar.

ParametreAnlamları
1. Parametre = Tablo AdıTablo adı.
2. Parametre = Kayıt Numarasıİstenilen kayıtın numarası.

Record::delete('uyeler', 5);  // 5 Numaralı veriyi sil.
NOTE
Tüm kayıtları silmek istenirse id değeri belirtilmez.
Record::delete('uyeler');  // Tüm verileri siler.

Veri Güvenliği Hakkında

Veriler dosyalara oluşturulduğu için URL üzerinden görünmesini engellemek için exit yönteminden yararlanılmıştır. Yani URL üzerinden oluşturacağınız verilerin görülmesi mümkün değildir. Ayrıca oluşturulan dosyalar .php uzantılı oluşturulduğundan uzaktan bağlantı ile indirilmeye de kapalıdır.

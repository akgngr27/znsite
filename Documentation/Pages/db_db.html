Hızlı Başlangıç ve Kullanım Örnekleri

Standart Sorgu Oluşturmak

Klasik sql sorgularını kullanmak için kullanılan bir yöntemdir.

$get = DB::query('select * from ornektablo');
echo $get->row()->id;
echo 'Toplam Satır:'.DB::totalRows();

ZN Özel Sorgu Oluşturmak

Özel yapılı sorgular oluşturmak için aşağıdaki gibi kullanımlar mümkündür.

$get = DB::get('ornektablo');

echo $get->row()->id;

Table() yöntemi ile tablo ismini belirleme; 
$get = DB::table('ornektablo')->get();

echo $get->row()->id;

$get = DB::where('id = ', 2)->get('ornektablo');
    
echo $get->row()->id;

$get = DB::select('id, isim')->where('id = ', 2)->get('ornektablo');
    
foreach($get->result() as $satir)
{
    echo $satir->id;
    echo $satir->isim;
}

Sorgu sonucu verileri dizi olarak elde etmek içinde aşağıdaki gibi bir kullanım yeterlidir.

$get = DB::where('id, isim')->select('id = ', 2)->get('ornektablo');
    
foreach($get->resultArray() as $satir)
{
    echo $satir['id']
    echo $satir['isim'];
}

Veri Eklemek

Veritabanına basit bir veri ekleme örneği.

DB::insert('ornektablo', array('id' => 2, 'isim' => 'zntr'));

DB::table('ornektablo')->insert(array('id' => 2, 'isim' => 'zntr'));

Veri Güncellemek

Veritabanına basit bir veri güncelleme örneği.

DB::where('id' => 1)->update('ornektablo', array('id' => 2, 'isim' => 'zntr'));

DB::where('id' => 1)->table('ornektablo')->update(array('id' => 2, 'isim' => 'zntr'));

Standart sorgu oluşturmak için query() yöntemi kullanılır.

query()
ParametreKullanımı
string SorguSorgu ifadesi

$get = DB::query('select * from ornektablo');
echo $get->row()->id;
echo 'Toplam Satır:'.$this->totalRows();

DB::query('delete from ornektablo where id = 1');

Tek Seferlik Çalıştırılabilir Sorgu Oluşturmak

DB::execQuery( string $sorgu )

Basit bir sorgu oluşturmak için execQuery() yöntemi kullanılır. Genellik bu yöntem çıktı üretmeyen sorgular yazılacağı zaman kullanılır.

execQuery()
ParametreKullanımı
string SorguSorgu ifadesi

DB::execQuery('create table ornektablo');
DB::execQuery('drop table ornektablo');

Veritabanından Veri Çekmek

DB::get( [ string $tablo_adi ] )

Veritabanından veri çekmek için kullanılır. Ayrıca oluşturulan özel yapılı sorguları sonlandırmak için kullanılır.

get()
ParametreKullanımı
[string Tablo Adı]from() yöntemi ile tablo ismi belirtilmemişse bu parametre ile tablo ismi belirtilebilir.

$get = DB::get('ornektablo');

$get = DB::table('ornektablo')->get();

Sorgu Sonucu Verileri Kullanmak

$get->result([$tur = 'object']) // Sorgunun sonucunu object türde döndürür.

$get->resultArray() // Sorgunun sonucunu dizi türünde döndürür.

$get->resultJson() // Sorgunun sonucunu string json veri türünde döndürür.

$get->row([$yazdirilabilir = false]) // Sorgunun sonucunu tek bir satır kayıtı döndürür.

$get->value() // Sorgunun sonucunu ilk kayıtının ilk sütununa ait veriyi verir.

$get->columns() // Tabloya ait sütunları döndürür.

$get->totalRows([$limitsizKayitSayisi = false]) // Toplam kayıt sayısını döndürür.

$get->totalColumns() // Toplam sütun sayısını verir.

$get->columnData() // Tabloya ait sütunlar hakkında veriler döndürür.

$get->fetch([$tur = 'assoc']) // standart fetch kullanımı ile aynıdır.

$get->fetchAssoc() // standart fetch assoc kullanımı ile aynıdır.

$get->fetchArray() // standart fetch array kullanımı ile aynıdır.

$get->fetchRow() // standart fetch row kullanımı ile aynıdır.

$get->tableName() // Tablo ismini döndüdür.

$get->affectedRows() // Etkilenen satır sayısını döndüdür.

$get->insertId() // Son eklenen Id'yi döndürür.

$get->pagination() // Sayfalama barını çıktılar.

$get->stringQuery() // Sorgunun metinsel değerini verir.

Veritabanından çekilen verileri object türünde kullanmak için result() yöntemi kullanılır.

$get = DB::get('ornektablo');

var_dump($get->result());

$get = DB::table('ornektablo')->get();

var_dump($get->result('array')); // resultArray() ile aynı işlevi görür

Verileri dizi türünde kullanmak için resultArray() yöntemi kullanılır.

$get = DB::get('ornektablo');

var_dump($get->resultArray());

Tek bir satır veriyi kullanmak için row() yöntemi kullanılır.

$get = DB::get('ornektablo');

echo $get->row()->id; // İlk kayıta ait id bilgisini döndürür.

echo $get->row(-1)->id; // Son kayıta ait id bilgisini döndürür.

echo $get->row(1)->id; // İkinci kayıta ait id bilgisini döndürür.

echo $get->row(2)->id; // Üçüncü kayıta ait id bilgisini döndürür.

Satır sayısı almak için totalRows(), imit varsa limit değerine rağmen toplam satır sayısı isteniyorsa totalRows(true) kullanılmalıdır.

$get = DB::limit(0, 2)->get('ornektablo');

echo $get->totalRows(); // 2

echo $get->totalRows(true); // 10 limitten etkilenmeyen toplam satır sayısı.

Select, From ve Where Deyimlerini Kullanmak

DB::select( [ string $sutun_isimleri = '*' ] )

DB::from( string $tablo_adi )

DB::where( string $sutun_adi_kosul_ifadesi , string/numeric $deger , [ string $baglac ] )

DB::where( array($sutun_adi_kosul_ifadesi , string/numeric $deger , [ string $baglac ]) , ... , [ string $baglac ])

SELECT, FROM ve WHERE deyimlerini kullanmak.

$get = DB::select('id')
            ->from('ornektablo')
            ->get();
            
echo $get->row()->id;

$get = DB::select()  // Varsayılan parametresi *
            ->from('ornektablo')
            ->where('id = ', 2)
            ->get();
            
echo $get->row()->id;

Birden fazla where yapısı kullanmakta mümkündür.

$get = DB::select()  // Varsayılan parametresi *
            ->from('ornektablo')
            ->where('id = ', 2, 'or')
            ->where('id = ', 1)
            ->get();
            
echo $get->row()->id;

$get = DB::select()  // Varsayılan parametresi *
            ->from('ornektablo')
            ->where('id = ', 1, 'and')
            ->where('isim = ', 'zntr')
            ->get();
            
echo $get->row()->id;

Birden fazla where yapısını sıralı parametre şeklinde kullanmakta mümkündür.

$get = DB::select()  // Varsayılan parametresi *
            ->from('ornektablo')
            ->where
            (
                array('id = ', 2, 'or'),
                array('id = ', 1)
            )
            ->get();
            
echo $get->row()->id;

Birden fazla where yapısını tek parametrede dizi olarak kullanmanızda mümkündür. Bu kullanımın amacı where sorgularının değişkene aktarılabilmesini sağlamaktır.

$wheres = array
(
	array('id = ', 2, 'or'),
    array('id = ', 1)
)

$get = DB::select()  // Varsayılan parametresi *
            ->from('ornektablo')
            ->where($wheres)
            ->get();
            
echo $get->row()->id;

Eğer Where yapıları içinde belirtilen şartların parantezler içinde olması gerekirse whereGroup() yönteminden yararlanılır.

$get = DB::select()  // Varsayılan parametresi *
            ->from('ornektablo')
            ->whereGroup(array('id = ', 1, 'or'), array('isim = ', 'zntr'), 'and')
            ->where('soyisim = ', 'deneme'))
            ->get();
            
echo $get->row()->id;

Where grubunu bir dizi değişkenine aktararak kullanmakta mümkündür.

$wheres = array
(
	array('id = ', 1, 'or'),
    array('isim = ', 'zntr')
)

$get = DB::select()  // Varsayılan parametresi *
            ->from('ornektablo')
            ->whereGroup($wheres, 'and')
            ->where('soyisim = ', 'deneme'))
            ->get();
            
echo $get->row()->id;

Veritabanına Veri Eklemek

DB::insert( string $tablo_adi , array $eklenecek_veriler )

INSERT deyimi kullanmak.

insert()
ParametreKullanımı
string Tablo AdıVerilerin ekleneceği tablo adı.
array Eklenecek VerilerTabloya eklenecek veriler.

DB::insert('ornektablo', array('id' => 1, 'isim' => 'zntr'));

Verilerin daha önce eklenip eklenmediğini kontrol etmek için duplicateCheck('*') yönteminden yararlanılabilir. Parametre yıldız ön tanımlıdır. Yıldız olursa tüm ekleme işlemindeki tüm belirtilen sütunlar kontrol edilir. Parametre olarak ekleme işleminde yer alan sütunlardan bir veya birkaçı da seçilebilir. Ancak bu kullanımda argümanlar sıralı olarak belirtilmelidir. Örnek: 's1', 's2' ...

DB::duplicateCheck()->insert('ornektablo', array('id' => 1, 'isim' => 'zntr'));

Eğer tablo ismi için table() yöntemi kullanılırsa yukarıdaki kodu aşağıdaki gibi düzenlemeniz gerekir.

DB::table('ornektablo')->insert(array('id' => 1, 'isim' => 'zntr'));

Aşağıdaki gibi bir kullanımda mümkündür.

DB::table('ornektablo')
  ->column('id', 1)
  ->column('isim', 'zntr')
  ->insert();

DB::table('ornektablo')
  ->column('id', 1)

Veri Güvenliğini Sağlamak

DB::secure( array $veriler)

Veritabanı işlemleri gerçekleştirilirken veri güvenliğini tehdit eden unsurların yok edilmesi amacıyla geliştirilmiştir.

DB::secure(array(':x' => 1))
     ->where('id = ', ':x')
     ->delete('ornektablo');

DB::secure(array(1, 'zntr'))
     ->where('id = ', '?', 'and')
     ->where('isim = ', '?')
     ->get('ornektablo');

DB::secure(array('secureId' => 1, 'secureName' => 'zntr'))
     ->insert('ornektablo', array('id' => 'secureId', 'isim' => 'secureName'));

Sütun Değerini İstenilen Miktarda Artırmak

DB::increment( string $tablo_adi, string/array $sutunlar, [ numeric $artis_miktari = 1] )

İstenilen sütun veya sütunların değerini istenilen miktarda artırmak için oluşturulmuştur.

DB::where('id = ', '1')
     ->increment('OrnekTablo', 'OrnekSutun');

table() yöntemi kullanılacak olursa increment() yönteminin sütunlar parametresi 1. parametre olur.

DB::where('id = ', '1')
     ->table('OrnekTablo')
     ->increment('OrnekSutun', 5);

Birden fazla sütunun değerini artırmak için aşağıdaki gibi bir kullanım söz konusu olur.

DB::where('id = ', '1')
     ->increment('OrnekTablo', array('OrnekSutun1', 'OrnekSutun2') );

DB::where('id = ', '1')

table() yöntemi kullanılacak olursa decrement() yönteminin sütunlar parametresi 1. parametre olur.

     ->table('OrnekTablo')

Birden fazla sütunun değerini artırmak için aşağıdaki gibi bir kullanım söz konusu olur.

Sayfalama(Pagination) Oluşturmak

DB::pagination( array $ayarlar, [ bool $donenVeriTuru = true] )

Sorgulara göre daha pratik bir sayfalama yapmak için oluşturulmuştur. Pagination sınıfının kullanımından yararlanılmıştır. İster Pagination sınıfına ayarları gönderin isterseniz doğrudan pagination yani sayfalama barını oluşturun. Yani veritabanı ile sayfalama sınıflarının entegrasyonu ile oluşturulmuş yöntemdir.

ParametreAnlamları
1. Parametre = string UrlSayfalama linklerinin bağlantı kuracağı link bilgisi.
2. Parametre = array AyarlarSayfalama için gönderilecek gerekli ayarlar. Bu ayarların neler olduğu Pagination sınıfında zaten belirtilmiştir. Örnek: links, countLinks.
3. Parametre = bool Çıktı trueBu parametre false ayarlamadığı sürece sayfalama barını çıktılayacaktır. False olma durumunda sayfalama için gerekli ayarları döndürecektir.

$pagination = DB::limit(URI::segment(-1), 5)->get('OrnekTablo')->pagination();
     
print($pagination)

DB::switchCase($anahtar, array($case1 => $value1, ...))CASE ELSE ENDCASE $anahtar THEN $case1 WHEN $value1 .... ELSE END
DB::ifElse($kosul, $dogru, $yanlis)IF ELSEIF($kosul) $dogru ELSE $yanlis
DB::count($sutun)COUNTCOUNT($sutun)
.
.
.

Bu fonksiyonların son parametrelerinin true olması durumunda doğrudan yazdırılabilir çıktı üretir.

echo DB::select(DB::sum('total', true))->get('OrnekTablo')->row(true);

Ayrıca as anahtarıda kullanıbilir bunun için son prametre true ise sondan bir önceki, değilse son parametre aşağıdaki gibi belirtilebilir.

echo DB::sum('total', 'as total')->get('OrnekTablo')->row(true);

Diğer Veritabanı Yöntemleri

Bu bölümde veritabanı kütüphanesinın diğer yöntemlerinin nasıl kullanıldığı ile ilgili anlatıma yer verdik.

Diğer Yöntemler
YöntemİşlevKullanım
duplicateCheck('*') Insert işleminde sütunların daha önce aynı değere sahip olup olmadığını kontrol eder. Parametre olarak sıralı argümen kabul edilir. Ön tanımlı değeri yıldızdır. Sütun isimleri belirtilecekse 's1', 's2', 's3' ... şeklinde belirtilebilir. // @params ...args => sütun isimleri.
 DB::duplicateCheck('s1', 's2', 's3');
// ... Sütunların Kontrolü ...
func() AVG, COUNT gibi matematiksel deyimlerin kullanılmasını sağlar. // @params array => fonksiyon ismi ve alacağı değerler.
 DB::func('avg', 'sayi', 'id');
// ... AVG(sayi, id) ...
column() INSERT, UPDATE, CREATE TABLE yöntemlerinde colon adı ve alacağı değeri belirtmek için kullanılabilir. // @params string => Sütun adı.
// @params mixed => Sütun değeri.
 DB::table('table')
->column('name', 'zeroneed')
->column('phone', '542')
->insert();
// ... AVG(sayi, id) ...
having() HAVING deyiminin kullanılmasını sağlar. // @params string => sütun bilgisi.
// @params mixed => değer.
 DB::having('avg(yas) > ', 30);
// ... HAVING AVG(yas) > 30 ...
join() JOIN deyiminin kullanılmasını sağlar. // @params string => tablo adı.
// @params string => karşılaştırma.
// @params string => birleştirme türü.
 DB::join('OrnekTablo', 'OrnekTablo.id = Tablo.id', 'left');
// ... LEFT JOIN OrnekTablo ON OrnekTablo.id = Tablo.id ...
groupBy() GROUP BY deyiminin kullanılmasını sağlar. // @params string => sütun adı.
 DB::groupBy('ID');
// ... GROUP BY ID ...
orderBy() ORDER BY deyiminin kullanılmasını sağlar. // @params string => sütun adı.
// @params string => sıralama türü.
 DB::orderBy('ID', 'desc');
// ... ORDER BY DESC ...
limit() LIMIT deyiminin kullanılmasını sağlar. // @params numeric => başlangıç indeks numarası.
// @params numeric => kaç satır veri alınacağı.
 DB::limit(0, 2);

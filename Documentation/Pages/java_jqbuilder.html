JQBuilder Kütüphanesi

Jquery'nin animasyon komutları kullanılarak oluşturulmuştur. Daha basit bir şekilde hareket işlemlerini gerçekleştirebilmeniz mümkündür.

Zincirleme Kullanımlı JQBuilder Parametreleri
Seçiçi BelirlemekJqueryr::selector(string $selector)
Özelliği BelirlemekJqueryr::property(string $property, [ string/array $params ] )
Hareketi Kuyruğa Ekleyecek Şekilde TamamlamakJqueryr::complete()
Hareketi OluşturmakJqueryr::create(arguments $complete1, $complete2 ... $completeN)

Zincirleme Kullanımlı Jquery İşlemleri

Jquery::create( [ arguments $complete1, $complete2 ... $completeN ] )

JQBuilder kütüphanesi ile ilgili kodun nasıl yapılacağına ait aşağıda örneklere yer verilmiştir. create() yöntemi gönderimi tamamlamak için kullanılan nihai yöntemdir.

Aşağıda olayın nasıl oluşturulacağına dair bir örneğe yer verilmiştir.

echo Script::open(); // Script tagı açıyor ve script jquery dosyasını dahil ediyor.

echo Jquery::selector('#button')
              ->property('click', JQ::callback('e', 'alert(1)'))
              ->create();
           
echo Script::close();// Script tagını kapatıyor.

// Kaynak Kod: $("#button").click(function(e){alert(1)});

echo Jquery::selector('#button')      
             ->property('bind', array('click', JQ::callback('e', 'alert(1)')))
             ->create();
// Kaynak Kod: $("#button").bind("click", function(e){alert(1)});

Aşağıdaki gibi bir kullanım da mümkündür. property() işlevinin 2. parametresi ile geri çağrım işlevi oluşturabilirsiniz. 

echo Jquery::selector('#button')
              ->property('bind', array('mouseover', JQ::func('e', 'alert(1)'))
              ->create();
// Kaynak Kod: $("#button").bind("mouseover", function(e){alert(1)});

Birden fazla özellik eklemek için aşağıdaki yöntemler tercih edilebilir.

echo Jquery::selector('#button')      
              ->property('bind', array('click', JQ::callback('e', 'alert(1)')))
              ->property('hide', 1000)
              ->create();
// Kaynak Kod: $("#button").bind("click", function(e){alert(1)}).hide(1000);

Ya da

->create(JQ::property('show', 1000), JQ::property('fadeIn', array('slow', 'linear')));
// Kaynak Kod: $("#button").click(function(e){alert(1)}).show(1000).fadeIn("slow");

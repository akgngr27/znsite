Encoding Karakter Seti Listesini Öğrenmek

charsetList() v1.2

Geçerli karakter setlerinin listesini verir.

var_dump(charsetList()); // Çıktı: Karakter liste dizisi

Farklı İşletim Sistemlerine Göre Satır Sonu Karakterini Kullanmak

eol(numeric $tekrar_sayisi = 1 ) v1.3

İşletim sistemlerindeki farklılıktan kaynaklanan \n, \r karakter kullanımlarından doğan problemleri kaldırmak için oluşturulmuştur.

NoParametrelerAnlamları
1numeric Tekrar SayısıKarakterlerin kaç kez tekrarlanacağı.

echo eol(1); // Çıktı: WIN \n\r , MAC \r , OTHER \n
echo eol(3); // Çıktı: WIN \n\r\n\r\n\r , MAC \r\r\r , OTHER \n\n\n

Metinsel İfadeyi Bölmek ve İstenen Parçaya Ulaşmak

divide( string $metin , [ string $bolunen_ifade = '|' ] , [ string/int $parca_numarasi = 0 ] )

Parametre olarak girilen metin bilgisini, belirlenen karaktere göre parçalamak ve istenilen parçayı almak için kullanılır.

NoParametrelerAnlamları
1string MetinBölmek istenilen metin.
2[ string Bölünen İfade = "|" ]Parçalara hangi karakteri kullanarak ayırsın.
3[ string/int Parça Numarası = 0 ]Parça numarası.
Parça Numarası Parametresinin Alabileceği Değerler
Sıfır veya Pozitif SayıBaştan parça numarası.
Negatif SayıSondan parça numarası.
lastSon parça numarası.
firstİlk parça numarası.
allTüm parçalar dizi olarak döndürür.

echo divide('bilgi|zntr|framework'); // Çıktı: bilgi
echo divide('bilgi-zntr-framework', '-', 1); // Çıktı: zntr
echo divide('bilgi-zntr-framework', '-', -1); // Çıktı: framework
echo divide('bilgi?zntr?framework', '?', 'last'); // Çıktı: framework
echo divide('bilgi+zntr+framework', '+', 'first'); // Çıktı: bilgi
echo divide('bilgi+zntr+framework', '+', 'all'); // Çıktı: array('bilgi', 'zntr', 'framework')

Son Oluşan Hata Bilgisini Öğrenmek

errorReport( string $hata_anahtari ) v1.1

Son oluşan hata bilgisini bir dizi olarak döndürür. Ancak istenirse parametre olarak girilen dizi anahtarlarından birine ait veriye ulaşılabilir.

NoParametrelerAnlamları
1string Hata AnatarıHata dizisinden istenilen hata bilgisini almak için kullanılır.

echo errorReport('message'); // Çıktı: Oluşan hata mesajını verecektir.
echo errorReport('line'); // Çıktı: Oluşan hata satır numarasını verecektir.
var_dump(errorReport()); // Çıktı: Oluşan hata dizisini verecektir.

Dil Dosyalarından Mesaj Çağrısı Yapmak

getErrorMessage( string $dil_dosyasi , string $mesaj_anahtari , string/array $degisiklik)

Herhangi bir dil dosyasından istenilen mesajı çağırmak için kullanılır.

NoParametrelerAnlamları
1string Dil Dosyasının AdıMesaj alınmak istenen dil dosyasının adı.
2string Mesah AnahtarıDil dosyasındaki istenilen mesaja ait anahtar.
1[ string/array Değişiklik ]Mesajda ye alan özel karakterler yerine istenilen kelimeleri kullanmak içindir.

echo getErrorMessage('Upload', 'upload_unknown_error'); // Çıktı: Bilinmeyen hata veya çok büyük boyutlu dosya!
echo getErrorMessage('Validation', 'validation_required'); // Çıktı: % alanı boş geçilemez!
echo getErrorMessage('Validation', 'validation_required', 'Text'); // Çıktı: Text alanı boş geçilemez!

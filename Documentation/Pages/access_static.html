Statik Erişim

Sınıflar ve import yöntemleri statik formda oluşturulduktan dolayı nesnelere statik olarak erişilebilmektedir.

Statik çağrı, performans açısından en verilimli çağrıdır.

Controllers/Anasayfa.php 
<?php
class Anasayfa extends Controller
{
	function index()
        {
            Import::page('Anasayfa');
        }
}

Statik formda tanımlanan her kütüphaneye statik olarak erişilebilmektedir.

Statik Olmayan(Non Static) Sınıfları Statik Erişim ile Kullanmak

Statik olmayan bir kütüphanesi hem statik hemde dinamik olarak kullanmak isterseniz bu kütüphanesinın isminin başına __USE_STATIC_ACCESS__ (Başında ve sonunda 2 alt tire bulunmaktadır.) ön ekini getirmeniz yeterlidir.

Aşağıda statik olmayan bir kütüphanesin nasıl statik kullanılabileceği hakkında örnek verilmiştir.

Libraries/Test.php 
<?php
class Test
{
	public function ornek()
        {
            // kodlar
        }
}

Yukarıdaki kütüphanesi dinamik olduğu kadar statik olarakta kullanmak isterseniz aşağıdaki gibi kullanmanız gerekmektedir.

Libraries/Test.php 
<?php
class __USE_STATIC_ACCESS__Test  Dikkat! __USE_STATIC_ACCESS__ ön eki getirilmiştir.
{
	public function ornek()
        {
            // kodlar
        }
}

Yukarıda sınıf isminin başına konulan __USE_STATIC_ACCESS__ ön eki sayesinde artık bu kütüphanesi statik olarak kullanabilirsiniz.

test::ornek();
// Ya da
$this->test->ornek();

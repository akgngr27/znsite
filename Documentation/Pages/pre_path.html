Sitenin Aktif Yolunu Öğrenmek

currentPath( [ boolean $tam_yol = true ] ) v1.0

Sitenin aktif yolunu öğrenmek için kullanılır.

NoParametrelerAnlamları
1[ boolean Tam Yol = true ]Aktif yolun tamamını mı yoksa son bölümünü mü versin.

echo currentPath(); // Çıktı: anasayfa/ornek/uygulama
echo currentPath(false); // Çıktı: uygulama

Sitenin Kök Yolunu Öğrenmek

basePath( [ string $yol_eki ] , [ numeric $dizin_indeksi = 0 ] ) v1.0

Sitenin kök yolunu öğrenmek için kullanılır.

NoParametrelerAnlamları
1string Yol EkiSite yoluna eklenecek ifadeler.
2[ string Dizin Indeksi ]Yolun üst dizinlerine ulaşabilmek için kullanılır.

echo basePath(); // Çıktı: zntr/
echo basePath('ornek/uygulama'); // Çıktı: zntr/ornek/uygulama
echo basePath('ornek/uygulama' , -1); // Çıktı: ornek/uygulama

Gelinen Bir Önceki Yolu Öğrenmek

prevPath( [ boolean $tam_yol = true ] ) v1.0

Ziyaret edilen sayfadan bir önceki sayfanın yolunu öğrenmek için kullanılır.

NoParametrelerAnlamları
1[ boolean Tam Yol = true ]Bir önceki yolun tamamını mı yoksa son bölümünü mü versin.

echo prevPath(); // Çıktı: zntr/ornek/uygulama
echo prevPath(false); // Çıktı: uygulama

URL Ekli Yolu Ham Yola Çevirmek

filePath( string $dosya_yolu , [ string $url_fakli_ise ] ) v1.0

Dosya yolu URL adresi içeriyorsa bu URL adresi temizleyerek ham yolu verir.

NoParametrelerAnlamları
1string Dosya YoluDosyanın yolu.
2[ string Farklı URL = "" ]Eğer dosya sitenin kök URL adresi dışında farklı bir URL bilgisi içeriyorsa kullanılır.

echo filePath( baseUrl('ornek/uygulama') ); // Çıktı: ornek/uygulama
echo filePath('http://www.siteadi.xxx/ornek/uygulama' , 'http://www.siteadi.xxx/'); // Çıktı: ornek/uygulama

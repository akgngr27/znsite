Metindeki Tüm Boşlukları Temizlemek

Strings::mtrim( string $metin )

Metinsel ifadelerdeki boşlukları temizlemek için kullanılır.

NoParametrelerAnlamları
1string MetinBoşlukların temizleneceği metin.

echo Strings::mtrim(' Zn framework '); /* Çıktı: Znframework */

Taksim (/) İşaretini Temizlemek

Strings::trimSlashes( string $metin )

Metinsel ifadelerde baştaki ve sonraki taksim işaretlerini temizlemek için kullanılır.

NoParametrelerAnlamları
1string MetinTaksim işaretlerinin temizleneceği metin.

echo Strings::trimSlashes('/urunler/bilgisayar/12/'); /* Çıktı: urunler/bilgisayar/12 */

Metinsel İfadelerde Aramak

Strings::search( string $metin , string $aranacak_kelime = 0 , [ string $tip = 'str' ] )

Metinsel ifadelerde kırpma işlemi yapmak için kullanılır.

NoParametrelerAnlamları
1string MetinArama işleminin yapılacağı metin.
2string Arancak KelimeAranacak ifade.
3[ string Çıktı Türü ]Çıktının türü. Varsayılan aranan metin.
3 Parametrenin Alabileceği Değerler
str veya stringArama sonucunda veri bulunmuşsa aranan veri döner.
pos veya positionArama sonucunda veri bulunmuşsa aranan verinin başlangıç indeks numarası döner.

echo Strings::search('Zntr kod çatısına hoş geldiniz.', 'tr'); /* Çıktı: tr */
echo Strings::search('Zntr kod çatısına hoş geldiniz.', 'tr', 'pos'); /* Çıktı: 2 */

Metinsel İfadelerde Karakterler Arası Yer Değiştirmek

Strings::reshuffle( string $metin , string $degisecek_ilk_karakter , string $degisecek_ikinci_karakter )

Metinsel ifadelerde istenilen karakter veya kelimeler arasında yer değiştirmek için kullanılır.

NoParametrelerAnlamları
1string MetinDeğişiklik işleminin yapılacağı metin.
2string Değişecek İlk KarakterYer değiştirmesi istenilen karakter.
3string Değişecek İkinci Karakterİlk karakterle yer değiştirmesi istenilen ikinci karakter.

echo Strings::reshuffle('a-b-c+d+e', '-', '+'); /* Çıktı: a+b+c-d-e */
echo Strings::reshuffle('Zntr kod çatısına hoş geldiniz.', 'kod', 'hoş'); /* Çıktı: Zntr hoş çatısına kod geldiniz. */

Metinsel İfadelerde Tekrarlanayan Metin Sayısını Hesaplamak

string_recurrentCount( string $metin , string $tekrarlanan_metin )

Metinsel ifadelerde tekrar eden metin sayısını öğrenmek için kullanılır.

NoParametrelerAnlamları
1string Metinİşleminin yapılacağı metin.
2string Tekrarlanan MetinTekrar sayısı öğrenilmek istenen metin.

echo Strings::recurrentCount('Merhaba kullanıcı', 'a'); /* Çıktı: 3 */
echo Strings::recurrentCount('Zntr kod çatısına hoş geldiniz.', 'kod'); /* Çıktı: 1 */

Metinsel İfadelerde Karakter Yerleştirmek

Strings::placement( string $metin , string $degistirilecek_metin , array $yeni_karakterler )

Bu yöntem str_replace() yönteminden farklı olarak değiştirilmek istenen karakter yerine sırası ile hangi karakterlerin geleceğini belirtir. Bununla ilgili aşağıda örnek kod yer almaktadır.

NoParametrelerAnlamları
1string Metinİşleminin yapılacağı metin.
2string Değiştirilecek KarakterMetin içinde hangi karakterin değiştirileceği.
3array Yeni KarakterlerDeğiştirilecek karakter yerine sırası ile hangi karakterleri yerleştirileceği.

echo Strings::placement('Merhaba kullanıcı', 'a', array('1', '2', '3')); /* Çıktı: Merh1b2 kull3nıcı */
echo Strings::placement('Zntr kod çatısına hoş geldiniz.', 'kod', array('KOD')); /* Çıktı: Zntr KOD çatısına hoş geldiniz.*/

Dizgenin Bir Bölümünü Almak

Strings::section( string $metin , numeric $baslangic , numeric $karakterUzunlugu , string $kodlamaTuru = 'utf-8')

Bu yöntem mb_substr() yönteminin kullanımı ile aynıdır.

NoParametrelerAnlamları
1string Metinİşleminin yapılacağı metin.
2numeric BaşlangıçMetnin kaçıncı indisten itibaren bölünmeye başlayacağı.
3numeric Karakter UzunluğuKaç karakter alınacağı. Herhangi bir uzunluk belirtilmezse dizgenin son karakterine kadar olan bölüm alınır.
4string Kodlama Türü = UTF-8Kodlama türünü ayarlamak için kullanılır. Varsayılan:utf-8

echo Strings::section('Merhaba kullanıcı', 8, 9); /* Çıktı: kullanıcı */

NoParametrelerAnlamları

Dizgenin Seçilen Yerlerini Değiştirmek

Strings::replace( string $metin , string $eskiKarakterler , string $yeniKarakterler , boolean $harfDuyarliligi = true )

Bu yöntem str_replace ve str_ireplace() yönteminin kullanımı ile aynıdır. Tek farklılık parametredeki dizilimdir. Bu yöntemde 1. parametre değişliğin uygulanacağı dizge bilgisidir.

NoParametrelerAnlamları
1string Metinİşleminin yapılacağı metin.
2string Eski KarakterlerDeğiştirlemek istenen karakterler.
3string Yeni KarakterlerEski karakterlerin yerine gelecek yeni karakterler.
4bool Harf Duyarlılığı = trueBüyük - küçük harf duyarlılığının olup olmayacağı. Varsayılan: true

echo Strings::replace('Merhaba kullanıcı', 'kullanıcı', 'dünya'); /* Çıktı: Merhaba dünya */
echo Strings::replace('Merhaba kullanıcı', 'KULLANICI', 'dünya', false); /* Çıktı: Merhaba dünya */

Dizgeyi Diziye Çevirmek

Strings::toArray( string $metin , string $ayrac )

Bu yöntem dizgeyi 2. parametre belirtilmezse boşluk karakterlerine göre diziye çevirir.

NoParametrelerAnlamları
1string Metinİşleminin yapılacağı metin.
2string AyraçDizgenin hangi karakterden itibaren diziye çevireleceği. Varsayılan: boşluk karakteridir.

output( Strings::toArray('Merhaba kullanıcı') ); /* Çıktı: array('Merhaba', 'kullanıcı') */
output( Strings::toArray('Merhaba-kullanıcı', '-') ); /* Çıktı: array('Merhaba', 'kullanıcı') */

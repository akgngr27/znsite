Compress(Sıkıştırma) Sürücüleri

Sıkıştırma işlemlerini gerçekleştirmek üzere oluşturulmuş. Ön tanımlı sıkıştırma Config/Compress.php ayar dosyasında yer alan driver bölümnde gz olarak tanımlanmıştır.

Compress(Sıkıştırma) Sürücüleri
Desteklenen Sürücülerbz, gz, lzf, rar, zip, zlib
YöntemlerYöntemi Destekleyen Sürücüler
Extract -> Dosyadan çıkartmarar, zip
Write -> Dosyaya veri yazdırmabz, gz
Read -> Dosyadan veri okumabz, gz, zip
Compress -> Dizgeyi sıkıştırmabz, gz, lzf
Uncompress -> Sıkıştırılan dizgeyi açmabz, gz, lzf
Encode -> Dizgeyi şifrelemegz, zlib
Decode -> Şifrelenen dizgeyi çözmegz, zlib
Deflate -> Deflate yöntemi ile sıkıştmagz
Inflate -> Deflate yöntemi ile sıkıştırılmış veriyi açmagz

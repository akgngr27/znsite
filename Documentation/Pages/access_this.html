Yöntemsel Erişim

Nesnelere erişim sağlayabilen yöntemlerden biride yöntemsel çağrıdır.

Performans açısından en yavaş çağrıdır. Gerek kalmadıkça kullanmaktan kaçınmanız önerilir.

Controllers/Anasayfa.php 
<?php
class Anasayfa extends Controller
{
	function index()
        {
            echo uselib('Encode')->super('Veriyi Şifrele');
        }
}

Yöntemsel çağrı, Controllers extends'i gerektirmeden her sayfadan erişim imkanı sağlar.

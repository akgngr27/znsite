Script Kütüphanesi

Bu kütüphane tamamen JSuery kütüphanelerine yardımcı parçalar sağlamak için oluşturulmuştur. JSuery'ye ait bazı kod parçaları gerekli olduğunda bu ihtiyacı gidermek için bu kütüphaneden yararlanılabilir. Örnek olarak .show(); gibi bir kod parçası lazım olduğunda bu kütüphaneden yararlanılır. Ya da $(xx).serialize() gibi bir veri gönderimi için de yine bu kütüphaneden yararlanılır.

JS Kütüphanesi Yöntemleri
Script Tagı AçmakScript::open(bool $ready = true)
Script Tagı KapatmakScript::close()
Script Türünü AyarlamakScript::type(string $type = 'text/javascript')
Harici Script Kütüphaneleri Dahil EtmekScript::library(arguments ... )

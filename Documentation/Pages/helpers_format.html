Byte Biçimi Vermek

Format::byte( numeric $formatlanacak_byte_miktari , [ numeric $ondalik = 1 ] , [ boolean $birim_goster = true ] )

Girilen bayt miktarını biçimlendirir.

NoParametrelerAnlamları
1string/int Fortmalanacak Byte MiktarıBayt miktarı.
2[ string/int Ondalık = 1 ]Biçimlendirme sonrası oluşan yeni ifadenin virgülünden sonraki basamak sayısı.
3[ boolean Birim Gösterilsin Mi?]Birim gösterilsin mi? False değeri olması durumnda birim gösterilmeyecektir.

echo Format::byte(1231324); /* Çıktı: 1.2 MB */
echo Format::byte(3453454353, 3); /* Çıktı: 3.216 GB */
echo Format::byte(568567655465476575, 2, false); /* Çıktı: 504.99 */

Para Biçimi Vermek

Format::money( numeric $formatlanacak_para_miktari , [ string $para_birimi ] )

Girilen sayısal değeri para biçimine çevirir.

NoParametrelerAnlamları
1string/int Fortmalanacak Para MiktarıPara miktarı.
2[ string Para Birimi ]Biçimlendirme sonrası oluşan yeni değerin sonuna eklenecek para birimi.

echo Format::money(1231324); /* Çıktı: 1.231.324,00 */
echo Format::money(3453454353, 'TL'); /* Çıktı: 3.453.454.353,00 TL */
echo Format::money(568567655465, '£'); /* Çıktı: 568.567.655.465,00 £ */

Sayısal Zaman Verilerini Dönüştürmek

Format::time( numeric $formatlanacak_saniye_miktari , [ string $giris_tipi = 'day' ] , [ string $cikis_tipi = 'second' ] )

Girilen sayısal değeri bir türden diğer türe çeviren araçtır.

NoParametrelerAnlamları
1string/int Fortmalanacak Zaman MiktarıZaman miktarı.
2[ string Giriş Tip = "day"]1. parametrenin ne cinsinden ifade edileceği.
2. Parametrenin Alabileceği Değerler
secondSaniye.
minuteDakika.
hourSaat.
dayGün.
monthAy.
yearYıl.
3[ string Çıktı = "second"]Çıktının ne cinsinden ifade edileceği.
3. Parametrenin Alabileceği Değerler
secondSaniye.
minuteDakika.
hourSaat.
dayGün.
monthAy.
yearYıl.

echo Format::time(90 , 'second' , 'minute'); /* 90 Saniyeyi Dakikaya Çevir Çıktı: 1.5 */
echo Format::time(30 , 'day' , 'month'); /* 30 Günü Aya Çevir Çıktı: 1 */
echo Format::time(12 , 'month' , 'year'); /* 12 Ayı Yıla Çevir Çıktı: 1 */

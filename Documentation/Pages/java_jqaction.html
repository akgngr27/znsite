JQAction Kütüphanesi

Jquery'nin animasyon komutları kullanılarak oluşturulmuştur. Daha basit bir şekilde hareket işlemlerini gerçekleştirebilmeniz mümkündür.

Zincirleme Kullanımlı JQAction Parametreleri
Seçiçi BelirlemekJquery::action()->selector(string $selector)
Hızı AyarlamakJquery::action()->duration/speed(string/numeric $speed)
Easing Efekti UygulamakJquery::action()->easing(string $type)
Hareket Efekti SeçmekJquery::action()->type(string $type)
Geri Dönüş Fonksiyonu KullanmakJquery::action()->callback/func(string $parameter, string $code)
Hareketi Kuyruğa Ekleyecek Şekilde TamamlamakJquery::action()->complete()
Hareketi OluşturmakJquery::action()->create(arguments $complete1, $complete2 ... $completeN)

Tekil Kullanımlı JQAction Hareket Yöntemleri
Show Efektini KullanmakJquery::action()->show(string $selector, mixed $timing, string $callback)
Hide Efektini KullanmakJquery::action()->hide(string $selector, mixed $timing, string $callback)
Fade In Efektini KullanmakJquery::action()->fadeIn(string $selector, mixed $timing, string $callback)
Fade Out Efektini KullanmakJquery::action()->fadeOut(string $selector, mixed $timing, string $callback)
Fade To Efektini KullanmakJquery::action()->fadeTo(string $selector, mixed $timing, string $callback)
Slide Up Efektini KullanmakJquery::action()->slideUp(string $selector, mixed $timing, string $callback)
Slide Down Efektini KullanmakJquery::action()->slideDown(string $selector, mixed $timing, string $callback)
Slide Toggle Efektini KullanmakJquery::action()->slideToggle(string $selector, mixed $timing, string $callback)

Zincirleme Kullanımlı Jquery Hareket İşlemleri

Jquery::action([$scriptOutput = false])->create( [ arguments $complete1, $complete2 ... $completeN ] )

JQAction kütüphanesi ile ilgili hareketin nasıl yapılacağına ait aşağıda örneklere yer verilmiştir. create() yöntemi gönderimi tamamlamak için kullanılan nihai yöntemdir.

Aşağıda hareketin nasıl oluşturulacağına dair bir örneğe yer verilmiştir.

echo Script::open(); // Script tagı açıyor ve script jquery dosyasını dahil ediyor.

echo Jquery::action()->selector('#button')
             ->type('hide')
             ->speed(1000)
             ->callback('e', 'alert(1)')
             ->create();
           
echo Script::close();// Script tagını kapatıyor.

// Kaynak Kod: $("#button").hide(1000,function(e){alert(1)});

Harekete kuyruk eklemek için complete() yöntemini aşağıdaki gibi düzenlemeniz yeterlidir.

->create(JQ::property('show', 1000), JQ::property('fadeIn', array('slow', 'linear')));
// Kaynak Kod: $("#button").hide(1000,function(e){alert(1)}).show(1000).fadeIn("slow");

Tekil Kullanımlı Hareket Yöntemlerinin Kullanımı

Jquery::action()->show('#button', 1000, 'alert(1)');
// Kaynak Kod: $("#button").show(1000,function(e){alert(1)});

Jquery::action()->fadeIn('#button', 1000, 'alert(1)');
// Kaynak Kod: $("#button").fadeIn(1000,function(e){alert(1)});

Zincirleme Kullanımlı Jquery Olay İşlemleri

Jquery::event([$scriptOutput = false])->create( [ arguments $complete1, $complete2 ... $completeN ] )

JQEvent kütüphanesi ile ilgili olayların nasıl yapılacağına ait aşağıda örneklere yer verilmiştir. create() yöntemi gönderimi tamamlamak için kullanılan nihai yöntemdir.

Aşağıda olayın nasıl oluşturulacağına dair bir örneğe yer verilmiştir.

echo Script::open(); // Script tagı açıyor ve script jquery dosyasını dahil ediyor.

echo Jquery::event()->selector('#button')
             ->type('click')
             ->callback('e', 'alert(1)')
             ->create();
           
echo Script::close();// Script tagını kapatıyor.

// Kaynak Kod: $("#button").click(function(e){alert(1)});

Harekete kuyruk eklemek için complete() yöntemini aşağıdaki gibi düzenlemeniz yeterlidir.

->create(JQ::property('show', 1000), JQ::property('fadeIn', array('slow', 'linear')));
// Kaynak Kod: $("#button").click(function(e){alert(1)}).show(1000).fadeIn("slow");

İstenirse bind(), on() gibi yöntemlerde kullanılabilir. 

echo Jquery::event()->selector('#button')
             ->on('click', '{ foo: "bar" }', JQ::func('alert("Olay Gerçekleşiyor")'))
             ->create();
             
// Kaynak Kod: $("#button").on('click', { foo: "bar" }, function(e){alert("Olay Gerçekleşiyor")});

Tekil Kullanımlı Olayların Kullanımı

Jquery::event()->click('#button', 'alert(1)');
// Kaynak Kod: $("#button").click(function(e){alert(1)});

Jquery::event()->scroll('#button', 'alert(1)');
// Kaynak Kod: $("#button").scroll(function(e){alert(1)});

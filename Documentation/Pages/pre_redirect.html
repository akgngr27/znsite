Yönlendirme Yapmak

redirect( string $url_yada_uri , [ numeric $saniye = 0 ] , [ array $data ] , [ boolean $cikis = true ] ) v1.0

Yönlendirme yapmak için kullanılır.

NoParametrelerAnlamları
1string URL veya URIYönlendirilmek istenen adres veya yol.
2[ string Yönlendirme Bekleme Süresi = 0 ]Yönlendirmenin kaç saniye sonra yapılacağı.
3[ array Gönderilecek Veriler = array() ]Yönlendirilen sayfa veri taşımak için kullanılır.
4[ boolean Çıkış = true ]Yönlendirme yapılırken sayfa kodları okunsun mu okunmasın mı. Önerilen değer: true.

echo redirect('http://www.siteadi.xxx/ornek/uygulama');
echo redirect('ornek/uygulama');
echo redirect('ornek/uygulama', 1); // 1 saniye sonra yönlendirme yap.
echo redirect('ornek/uygulama', 0, array('bilgi' => 'zntr') ); // Yönlenen adrese bilgi verisi taşı.

Yönlendirmeden Gelen Veriyi Kullanmak

redirectData( string $veri_adi ) v1.0

Yönlendirmeden gelen veriyi kullanmak için oluşturulmuş bir yöntemdir. Yönlendirme foksiyonu ile gönderilen veriler dizi şeklinde gönderilmektedir ve bilgiler oluşturulurken anahtar-değer çifti kullanılır. Bu yöntem, yönlendirme yapılırken gönderilen bilginin anahtar kelimesini kullanarak gönderilen bilgiye erişim sağlar.

NoParametrelerAnlamları
1string Gelen Veri AdıYönlendirme yapılırken gönderilen verinin adı.

echo redirectData('bilgi'); // Çıktı: zntr

Yönlendirmeden Gelen Veriyi Kullanmak

redirectDeleteData( string/array $veri_adi ) v2.0

Yönlendirmeden gelen veriyi silmek için oluşturulmuş bir yöntemdir. Tek bir veri silinecekse metinsel aynı anda birden fazla veri silinecek dizisel olarak parametre gönderilebilir.

NoParametrelerAnlamları
1string/array Gelen Veri AdıYönlendirme yapılırken gönderilen verinin adı.

echo redirectDeleteData('bilgi');
echo redirectDeleteData(array('bilgi', 'bilgi2'));

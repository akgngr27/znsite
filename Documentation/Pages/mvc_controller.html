Controller(Kontrolcü) Kullanımı

ZN kod çatısında tasarım sayfaları, Controllers kontrolcü sayfaları tarafından yönetilir. ZN kod çatısında URL adresi Controllers kodlayıcı sayfalarının içerisindeki sınıf ismini ve o sınıfa ait fonksiyon ve parametre bilgilerini içerecek şekilde kullanılır.

ornek.com/index.php/blog/fonksiyon/parametre

Yukarıdaki yol, Controllers/ dizini içerisindeki blog.php sayfasını ifade etmekte iken fonksiyon, blog.php sayfası içerisindeki çalıştırılmak istenen herhangi bir yöntemi ifade etmektedir.

Basit Bir Kod Çalıştırmak

Aşağıda "ZN Kod Dünyasına Hoş Geldiniz." iletisini veren örnek bir kod yer almaktadır.

<?php
class Blog extends Controller
{
	public function index()
        {
            echo "ZN Kod Dünyasına Hoş Geldiniz.";
        }
}

Şimdi de bu kodu çalıştırmak için adres çubuğuna aşağıdaki URL adresini girin.

ornek.com/index.php/blog/index

NOT
Dosya ismi küçük harfle başlasa bile sizin sınıf adını, büyük harfle başlayacak şekilde yazmanız tavsiye edilir. Örnek:Sayfa adı "blog.php", sınıf adı "Blog" gibi.

Fonksiyon Bölümü(Segmenti)

Adres çubuğuna index.php'den sonra birinci bölüme erişilecek dosya/sınıf adı, ikinci bölüme ise o sınıfa ait çalıştırılacak fonksiyonun adı yazılır. Aşağıdaki URL adresinde 2. bölüm ise index fonksiyonun adıdır.

ornek.com/index.php/blog/index

Aşağıdaki kodda iki adet fonksiyon yer almaktadır.

<?php
class Blog extends Controller
{
        public function index()
        {
                echo 'ZN Kod Dünyasına Hoş Geldiniz.;
        }

        public function yorumlar()
        {
                echo 'Şimdi de beni çalıştır!';
        }
}

Yukarıda yer alan yorumlar fonksiyonunu çalıştırmak için URL adresimizi şu şekilde düzenliyoruz.

ornek.php/index.php/blog/yorumlar

Eğer fonksiyon ismi main veya index ise URI de belirtmeye gerek yoktur. Yani yukarıdaki URL'yi şöylede yazabilirdik.

ornek.php/index.php/blog // çalışatırılan fonksiyon index'tir.

Parametreler Bölümü(Segmenti)

URL üzerinden fonksiyonlara pameretre göndermek için URL adresinin 3. ve sonraki bölümleri kullanılır. Aşağıdaki URL parametrik değerler içermektedir.

ornek.com/index.php/urunler/bilgisayar/fiyat/1000

Yukarıdaki URL adresinde parametrik değerler fiyat ve 1000'dir. Peki kod içerisinde bu değerleri nasıl kullanırız?

<?php
class Urunler extends Controller
{
        public function bilgisayar($fiyat, $deger)
        {
                echo $fiyat." : ".$deger; // Çıktı: fiyat : 1000
        }

}

Varsayılan Controllers Açılış Sayfası

ZN kod çatısında varsayılan olarak ayarlanmış açılış sayfası Controllers/home.php sayfasıdır. Bunu değiştirmek için Config/Route.php dosyasını açıp aşağıdaki ayarı değiştirmeniz yeterlidir.

$config['Route']['openPage'] = 'home';

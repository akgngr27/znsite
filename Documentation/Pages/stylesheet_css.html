CSS Kütüphanesi

CSS kullanımına yönelik yöntemler içermektedir.

CSS Kütüphanesi Yöntemleri

CSS kütüphanesine ait yöntemler aşağıdaki tabloda listelenmiştir

CSS Kütüphanesi ve YöntemleriAnlamlarıKullanımları
selector( $erisim )Nesneye erişim sağlamak için kullanılan adlandırmalar. Örn: .sinif, #id ...->selector('Erişim Adı')
attr( $ozellik_deger_dizisi )Eklenecek özellikler ve değerleri. Bu yöntem dizi parametresi içerir.->attr(array('border' => 'solid 1px #000'));

Yukarıdaki yöntemlerin kullanımına yönelik örnekler aşağıda verilmiştir.

Sheet::manipulation()->selector('#nesne')->attr(array('border' => 'solid 1px #000', 'color' => 'red'));

CSS Dosyasına Müdahale Etmek

CSS stil dosyalarına müdahale etmek içeriğini manipüle etmek için kullanılır. Böylece bir .css uzantılı dosyaları PHP ile değiştirebilirsiniz.

Manipülasyon YöntemleriAnlamlarıKullanımları
file( $dosya )Resources/Styles/ dizininde yer alan css dosyasının ismi belirtilir. Uzantı yazılmasına gerek yoktur. Örn: .sinif, #id ...->file('stil')
getSelector( $secici_adi )Stil dosyası içerisinde yer alan kodlardan hangi seçiciye ait olanlar isteniliyorsa o seçicinin adı yazılır.->getSelector(.red-color');
setSelector( $secici_adi , $yeni_degerler_dizisi )Dosya içerisinde belirtilen seçicinin kodlarını değiştirmek için kullanılır.->setSelector( 'secici' , array('border' => 'solid 1px #000', 'color' => 'red'));

Yukarıdaki yöntemlerin kullanımına yönelik örnekler aşağıda verilmiştir.

Sheet::manipulation()->file('stil')->setSelector('.red-color', array('border' => 'solid 1px #000', 'color' => 'red'));

Sheet::manipulation()->file('stil')->getSelector('.red-color'); // border:solid 1px #000; color:red;

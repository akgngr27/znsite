Karakter Dönüştürmek

Convert::char( string $metin , [ string $metnin_karakter_tipi = 'char' ] , [ string $metnin_donusecegi_karakter_tipi = 'html' ] )

Karakterleri html, decimal veya hexdecimal karakter formatına dönüştürmek için kullanılır.

NoParametrelerAnlamları
1string MetinDeğişiklik yapılacak metin.
2[ string Metnin Karakter Tipi = "char" ]Metnin karakter tipi. Varsayılan: char
3[ string Metnin Dönüşeceği Karakter Tipi = "html" ]Metnin dönüştürüleceği karakter tipi. Varsayılan: html
NoDönüşüm Yapılacak Karakter Tipleri
1char
2html
3dec
4hex

/* --- Char'dan Diğer Tiplere Dönüştürme --- */
echo Convert::char('Metin'); /* Kaynak Kod Çıktı: &#77;&#101;&#116;&#105;&#110; */
echo Convert::char('Metin', 'char', 'dec'); /* Çıktı: 77 101 116 105 110 */
echo Convert::char('Metin', 'char', 'hex'); /* Çıktı: 4D 65 74 69 6E */

/* --- Kendi Aralarında Dönüştürme --- */
$html = Convert::char('Metin');
$dec = Convert::char('Metin', 'char', 'dec');
$hex = Convert::char('Metin', 'char', 'hex');

echo Convert::char($hex, 'hex', 'char'); /* Çıktı: Metin */
echo Convert::char($dec, 'dec', 'hex'); /* Çıktı: 4D 65 74 69 6E */
echo Convert::char($html, 'html', 'dec'); /* Çıktı: 77 101 116 105 110 */

NoParametrelerAnlamları

Metinlerde Büyük Küçük Harf Dönüştürmek

Convert::stringCase( string $metin , [ string $tip = 'lower' ] , [ string $karakter_kodlamasi = 'utf-8' ])

Büyük metinsel ifadeleri küçük, büyük veya sadece ilk harfini büyük harfe dönüştürmek için kullanılır.

NoParametrelerAnlamları
1string MetinDeğişiklik yapılacak metin.
2[ string Tip = "lower" ]Dönüştürme işleminin türü varsayılan: lower(küçük harf)'dir.
Tip Parametresinin Alabileceği DeğerlerAnlamları
lowerKüçük harfe çevirir.
upperBüyük harfe çevirir.
titleSadece ilk harfleri büyük harfe diğerlerini küçük harfe çevirir.

echo Convert::stringCase('Zn Kod Çatısına Hoş Geldiniz'); /* Çıktı: zn kod çatısına hoş geldiniz */
echo Convert::stringCase('Zn Kod Çatısına Hoş Geldiniz', 'upper'); /* Çıktı: ZN KOD ÇATISINA HOŞ GELDİNİZ */
echo Convert::stringCase('Zn Kod Çatısına Hoş Geldiniz', 'title'); /* Çıktı: Zn Kod Çatısına Hoş Geldiniz */

Dizilerde Büyük Küçük Harf Dönüştürmek

Convert::arrayCase( array $array , [ string $tip = 'lower' ] , [ string $anahtarDeger = 'all' ])

Dizilerin anahtar ve değerlerinde büyük-küçük harf dönüşümü yapmak için kullanılır.

ParametrelerAnlamları
1 - string MetinDeğişiklik yapılacak metin.
2 - [ string Tip = "lower" ]Dönüştürme işleminin türü varsayılan: lower(küçük harf)'dir.
Tip Parametresinin Alabileceği DeğerlerAnlamları
lowerKüçük harfe çevirir.
upperBüyük harfe çevirir.
titleSadece ilk harfleri büyük harfe diğerlerini küçük harfe çevirir.
3 - [ string Tip = "all" ]Dönüştürme işleminin anahtara mı, değere mi yoksa her ikisine mi uygulanacak bu belirtilir. Varsayılan:all
allHem anahtara hem değere uygula.
keySadece anahtara uygula.
val/valueSadece değere uygula.

echo Convert::arrayCase(array('Merhaba' => 'ZN'), 'lower'); /* Çıktı: array('merhaba' => 'zn') */

Metni Sabit Olarak Kullanmak

Convert::toConstant( string $sabit, [ string $onek ], [ string $sonek ] )

Bu yöntemin temel amacı string veriyi bir sabitmiş gibi kullanabilmektir. Bir çok sabit konusuna göre ön ek içerir mesela MB_CASE_ veya CURL_INFO_ gibi. Bu şekilde uzun kullanımlar yerine sabitin metinsel isminin bir bölümü kullanılarak aynı değere ulaşmak mümkündür. Şimdi aşağıdaki örnekle bu yöntemin işlevi daha net anlaşılacaktır.

ParametrelerAnlamları
1 - string MetinSabitin strin adı veya bir bölümü.
2 - [ string Metin ]Ön ek.
3 - [ string Metin ]Son ek.

Convert::toConstant('upper', 'MB_CASE_'); // Return MB_CASE_UPPER
Convert::toConstant('autoreferer', 'CURLOPT_'); // Return CURLOPT_AUTOREFERER

Bu yöntem sayesinde sabiti string türde kullanmanız gerektiği zaman yararlanabilirsiniz. ZN Framework'te genellikle tanımlanan yöntemlerde sabit değer içermesi gereken parametreler de bu yöntemden yararlanılmıştır. Böylece sabitin tam adını yazmak yerine öne ekini atıp daha kısa bir şekilde kullanımı sağlanmıştır.

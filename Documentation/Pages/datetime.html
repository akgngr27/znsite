Date(Tarih) Kütüphanesi

Tarih ve Zaman ile ilgili işlevleri yerine getiren bir sınıftır. Bu kütüphane standart tarih ve saat formatlarının dışında Config/Datetime.php ayar dosyasında yer alan özel çevrimleride kullanmaktadır. Böylece daha akılda kalıcı tarih ve zaman terimleri kullanılabilir hale getirilmiştir. Bu bölümde Date ve Time olmak üzere iki kütüphanenin anlatımına yer verilmiştir. Her iki kütüphanenin özünde yaptığı işler aynı olsada karakter setlerinde ve local ayarlamalarında küçük farklılıklar bulunmaktadır.

Tarih Bilgisi Oluşturmak

Date/Time::set(string $bicimler, [int $timestamp = time()]): return string

Tarih oluşturmada Config/DateTime.php dosyasından setDateFormatChars listesinde yer alan yeni karakterler kullanılır. Bunun yanında standart date karakterleride kullanılabilmektedir.

echo Date::set('{day}'); /* Çıktı: Sunday */
echo Date::set('{monNum}-{mon} {day}'); /* Çıktı: 2-February Sunday */
echo Date::set('{H024}:{min}:{sec}'); /* Çıktı: 15:46:52 */
echo Date::set('{dayNumber0}.{monthNumber}.{year} {hour024}:{minute}:{second}'); /* Çıktı: 08.2.2015 15:47:06 */

Tarih Bilgisinde Aritmatik İşlemler Gerçekleştirmek

Date/Time::calculate(string $girdi, string $islem, [string $cikti = {year}-{monthNumber}-{dayNumber}]): return string

Tarih bilgisi üzerinde ekleme çıkarma gibi işlemler gerçekleştirmek için kullanılmaktadır.

echo Date::calculate('{year}-{monthNumber}-{dayNumber}', '30 day'); /* Çıktı: Mevcut tarihe 30 gün eklenmiştir */
echo Date::calculate('01/12/2015', '-30 day', {dayNumber}-{monthNumber}-{year}); /* Çıktı: Belirtilen tarihten 30 gün çıkartılıp çıktı türü ayarlanmıştır. */

Tarih Bilgilerini Karşılaştırmak

Date/Time::compare(string $deger1, string $karsilastirma, string $deger2): return bool

Tarih bilgilerini karşılaştırmak için kullanılır.

echo Date::compare('2015-02-15', '>', '2015-02-14'); /* Çıktı: Karşılaştırma doğru ise true değilse false değeri döner. */

Tarih Bilgisini Sayısal Veriye Dönüştürmek

Date/Time::toNumeric(string $tarih): return numeric

Tarih bilgisini sayısal veriye dönüştürmek için kullanılır.

echo Date::toNumeric('2015-02-15'); /* Çıktı: 1423951200 */

Tarih Formatını Değiştirmek

Date/Time::convert(string $tarih, string $format): return string

Tarih bilgisininin formatını değiştirmek için kullanılır.

echo Date::toNumeric('2015-02-15', '{dayNumber}/{monthNumber}/{year}'); /* Çıktı: 15/2/2015 */

Standart Tarih ve Zaman Bilgisini Almak

Date/Time::standart(): return string

Standart tarih ve zaman bilgisini almak için kullanılır. Eğer bu yöntemi Time sınıfı ile kullanılırsa bölge seçiminin diline göre çevrilir.

echo Date::standart(); /* Çıktı: 20.February.2016 Saturday, 12:46:55 */
echo Time::standart(); /* Çıktı: 20.Şubat.2016 Cumartesi, 12:46:55 */

Aktif Saat veya Tarih Bilgisini Almak

Date/Time::current(): return string

Bu yöntem Time sınıfı ile kullanılırsa aktif saat, Date sınıfı ile kullanılırsa aktif tarih bilgisini verir.

echo Date::current(); /* Çıktı: 20.02.2016 */
echo Time::current(); /* Çıktı: 12:52:45 */

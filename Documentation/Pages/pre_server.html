Ip ve Server Komutları

Server ve kullanıcı ip'si hakkında bilgi almak için kullanılan yöntemlerdir.

  • Ip ve Server Komutları

    • Ip Numarasını Öğrenmek » ipv4() v1.0

Ip Numarasını Öğremek

ipv4() v1.0

Kullanıcıların ip v4 adreslerini öğrenmek için kullanılır.

echo ipv4(); // Çıktı: 127.0.0.1

Server Verilerine Erişmek

server( string $veri_turu ) v1.0

$_SERVER Global değişkeninin işlevini üstlenir. Ancak server() yönteminin yazımı ve kullanımı biraz daha kolaydır.

IP Üzerinden DNS Kontrolü Yapmak

Net::checkDns( string $hostAdi/$IP, string $tur = 'MX' )

Belirtilen bir konak adı veya IP adresi için DNS sorgusu yapar. Kayıtlar bulunduğu takdirde TRUE, hiç kayıt yoksa veya bir hata oluşmuşsa FALSE döner.

ParametreAnlamları
1 - string Host Adı/IP AdresiKontrol edilecek host ismi.
2 - string Tür = MXSorgu türü. Kullanabilir türler:A, MX, NS, SOA, PTR, CNAME, AAAA, A6, SRV, NAPTR, TXT veya ANY. MX öntanımlıdır.
ReturnBool (Mantıksal TRUE, FALSE)

echo Net::checkDns('zntr.net');

Bir Hostla İlgili DNS Kayıtlarını Almak

Net::dnsRecords( string $hostAdi/$IP, string $tur = 'ANY' )

Belirtilen host adı ile ilgili DNS Özkaynak Kayıtlarını getirir.

ParametreAnlamları
1 - string Host Adı/IP AdresiKontrol edilecek host ismi.
2 - string Tür = anySorgu türü. Kullanabilir türler:A, MX, NS, SOA, PTR, CNAME, AAAA, A6, SRV, NAPTR, TXT veya ANY. MX öntanımlıdır.
ReturnObject (Nesne) -> records, -> authns, -> addtl

output( Net::dnsRecords('zntr.net') );

Bir Hostla İlgili MX Kayıtlarını Almak

Net::mxRecords( string $hostAdi )

Belirtilen host adı ile ilgili MX kayıtlarını döndürür.

ParametreAnlamları
1 - string Host AdıKontrol edilecek host ismi.
2 - string Tür = anySorgu türü. Kullanabilir türler:A, MX, NS, SOA, PTR, CNAME, AAAA, A6, SRV, NAPTR, TXT veya ANY. MX öntanımlıdır.
ReturnObject (Nesne) -> records, -> weight

output( Net::mxRecords('zntr.net') );

Internet Alan Soketi Bağlantısı Açmak

Net::socket( string $hostAdi, numeric $port = -1, numeric $timeout = 60)

Net::psocket( string $hostAdi, numeric $port = -1, numeric $timeout = 60)

Bir internet veya Unix alan soketi bağlantısı açar.

ParametreAnlamları
1 - string Host AdıBağlantı kurulacak host ismi.
2 - numeric Port NumarasıBağlantının sağlanacağı port.
3 - numeric Zaman AşımıZaman aşımı süresi.
ReturnResource (Kaynak)

$sock = Net::socket('zntr.net');
// Bağlantı ile ilgili bir hata oluşmuşssa Error Kütüphanesininden yararlanabilirsiniz.
output( Errors::getTable('Net', 'socket') );

// Kalıcı soket bağlantısı oluşturmak isterseniz psocket() yöntemi kullanılır.
$sock = Net::psocket('zntr.net');

IP Adresini Host İsmine Çevirmek

Net::ipv4ToHost( string $IP )

Belirtilen IP adresine çözümlenen host ismini döndürür. Sorgu başarısız olursa girilen IP değeri döner.

ParametreAnlamları

ReturnString (Dizge)

Belirtilen Ağ hizmeti ve Protokolle ilişkili Port Numarasını Almak

Net::servicePort( string $servis, string $protokol )

Belirtilen ağ hizmeti ve protokolle ilişkili port numarası ile döner.

ParametreAnlamları
1 - string Servis AdıServis adı. http, ftp, pop3, smtp, telnet, imap, www, gopher, finger
2 - string Protokol AdıProtokol adı. udp, tcp
ReturnInt (Sayısal)

output( Net::servicePort('http', 'tcp') ); // 80

2 - string Protokol AdıProtokol adı. udp, tcp
ReturnString (Dizge)

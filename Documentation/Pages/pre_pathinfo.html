Dosya ve Dosya Uzantıları Bilgisi Komutları

    • Dosya Uzantılarını Öğrenmek » extension() v1.0

Yol Bilgilerine Ulaşmak

pathInfos( string $dosya_yolu , [ string $bilgi = 'basename' ] ) v1.0

Yolu gösterilen dizin veya dosyaya ait çeşitli bilgilere ulaşmak için kullanılır.

NoParametrelerAnlamları
1string Dosya YoluDosyanın yolu.
2[ string Bilgi = "basename" ]Yol hakkında istenen bilgi türü.
2. Parametrenin Alabileceği Değerler
basenameDizin veya dosya yolunun son segmentini(bölümünü) verir.
dirnameYol bilgisine ait dizinin ismini verir.
filenameYol bilgisine ait dosyanın ismini verir.
extensionYol bilgisine ait dosyanın uzantısını verir.

echo pathInfos('ornek/uygulama/deneme.php'); // Çıktı: deneme.php
echo pathInfos('ornek/uygulama/deneme.php' , 'dirname'); // Çıktı: ornek/uygulama
echo pathInfos('ornek/uygulama/deneme.php' , 'filename'); // Çıktı: deneme
echo pathInfos('ornek/uygulama/deneme.php' , 'extension'); // Çıktı: php

Dosya Uzantılarını Öğrenmek

extension( string $dosya_ismi , [ boolean $nokta = false ] ) v1.0

Yol bilgisi olarak girilen dosyanın uzantısını verir.

NoParametrelerAnlamları
1string Dosya İsmiDosyanın adı.
2[ boolean Nokta = false ]Uzantının önünde nokta olsun mu olmasın mı.

echo extension('ornek/uygulama/deneme.php'); // Çıktı: php
echo extension('ornek.html'); // Çıktı: html
echo extension('ornek.html' , true); // Çıktı: .html

Dosya Uzantılarını Kaldırmak

removeExtension( string $dosya_ismi) v1.2

Yol bilgisi olarak girilen dosyanın uzantısını kaldırır.

NoParametrelerAnlamları
1string Dosya İsmiDosyanın adı.

echo removeExtension('ornek/uygulama/deneme.php'); // Çıktı: ornek/uygulama/deneme
echo removeExtension('ornek.html'); // Çıktı: ornek
echo removeExtension('ornek.h'); // Çıktı: ornek.h

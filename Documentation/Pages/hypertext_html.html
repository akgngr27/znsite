Eleman Yöntemi

Html::element( string $element , string $metin , [ array $ozellikler ] )

Herhangi bir html biçim etiketi uygulamak için kullanılır.

NoParametrelerAnlamları
1string ElementKullanılacak html etiketi.
2string MetinBiçimin uygulanacağı metin.
3[ array Özellikler ]Eklenecekse özellikler.

echo Html::element('u', 'Metin'); /* Kaynak Kod Çıktı: <u>Metin</u> */
echo Html::element('i', 'Metin'); /* Kaynak Kod Çıktı: <i>Metin</i> */
echo Html::element('strong', 'Metin', array('id' => 'metin')); /* Kaynak Kod Çıktı: <strong id="metin">Metin</strong> */

Çoklu Özellik Ekleme Yöntemi

Html::multiAttr( string $metin , [ array $bicimler ] )

Herhangi bir metne birden fazla biçim eklemek için kullanılır.

NoParametrelerAnlamları
1string MetinBiçimin uygulanacağı metin.
2[ array Biçimler ]Uygulanacak birden fazla biçim.

echo Html::multiAttr('Metin', array('u' , 'i' , 'b')); /* Kaynak Kod Çıktı: <u><i><b>Metin</b></i></u> */
// Özellik değer çifti eklemek isterseniz şöyle bir kullanımda mümkündür.
echo Html::multiAttr('Metin', array('u' , 'i' => array('id' = 2))); /* Kaynak Kod Çıktı: <u><i id="2">Metin</i></u> */

Başlık Yöntemi

Html::heading( string $metin , [ numeric $size = 3 ] , [ array $ozellikler ] )

Bir metni başlık bilgisi olarak düzenlemek için kullanılır.

NoParametrelerAnlamları
1string MetinBiçimin uygulanacağı metin.
2[ string/int Başlığın Boyutu = "3" ]Başlığın boyutu.
3[ array Özellikler ]Eklenecekse özellikler.

echo Html::heading('Metin'); /* Kaynak Kod Çıktı: <h3>Metin</h3> */
echo Html::heading('Metin', 1); /* Kaynak Kod Çıktı: <h1>Metin</h1> */
echo Html::heading('Metin', 2, array('id' , 'metin')); /* Kaynak Kod Çıktı: <h2 id="metin">Metin</h2> */

Meta Etiketi Yöntemi

Html::meta( string/array $isim , string $icerik , [ string $tip = 'name' ] )

Meta tag veya taglar eklemek için kullanılır.

NoParametrelerAnlamları
1string/array İsimEklenmek istenen isim. Author, description...
2string İçerikEklenmek istenen içerikler.
3[ string İsim Tipi = "name" ]name veya http değerleri alır.

echo Html::meta('author', 'Ozan Uykun'); /* Kaynak Kod Çıktı: <meta name="author" content="Ozan Uykun"> */
echo Html::meta('refresh', '20', 'http'); /* Kaynak Kod Çıktı: <meta http-equiv="refresh" content=20"> */

Çoklu meta tagı kullanmak isterseniz kodunuzu aşağıdaki gibi düzenliyorsunuz.

$metas = array(
        array('name' => 'author', 'content' => 'Ozan Uykun'),
        array('name' => 'refresh', 'content' => '20', 'type' => 'http')
);
echo Html::meta($metas);

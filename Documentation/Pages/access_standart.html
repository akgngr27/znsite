Standart Erişim

Nesnelere standar new anahtarı ile erişilen yöntemdir. ZN'nin kendi yöntemleri dışında new tanımlaması ile de nesnelere erişim sağlayabilirsiniz.

Controllers/Anasayfa.php 
<?php
class Anasayfa extends Controller
{
	function index()
        {
            $encode = New Encode;
            
            output( $encode->encode->super('Veriyi Şifrele'));
        }
}

Standart çağrı, Controllers extends'i gerektirmeden her sayfadan erişim imkanı sağlar.

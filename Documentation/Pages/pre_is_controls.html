PHP Sürüm Karşılaştırması Yapmak

isPhpVersion( string $surum_bilgisi = '5.2.4' ) v1.2

Parametre olarak yazılan sürümün desteklenip desteklenmediğini bilgisini true veya false çıktısı ile belirtir.

NoParametrelerAnlamları
1StringDesteklenme durumu öğrenilecek sürüm bilgisi.

echo isPhpVersion('5.6.0'); // Çıktı: false.

Bir Dosyanın Dahil Edilme Durumunu Kontrol Etmek

isImport( string $dosya_yolu ) v1.0

Herhangi bir dosyanın, require veya include yöntemlerinden herhangi biriyle daha önce dahil edilip edilmediği bilgisini verir. Herhangi bir dosya daha önce dahil edilmişse true değeri, edilmemişse false değeri döner.

NoParametrelerAnlamları
1Dosya YoluDahil edilip edilmediği kontrol edilecek dosyanın yolu.

echo isImport('OrnekDosya.php'); // Çıktı: false veya true

Bir Verinin URL Yapısına Uygunluğunu Kontrol Etmek

isUrl( string $url ) v1.1

Herhangi bir verinin, URL bilgisi olup olmadığını kontrol etmek amacıyla kullanılır.

NoParametrelerAnlamları
1string URLKontrol edilecek veri.

echo isUrl('www.zntr.net'); // Çıktı: false
echo isUrl('http://www.zntr.net'); // Çıktı: true

Bir Verinin E-posta Adresine Uygunluğunu Kontrol Etmek

isEmail(string $email ) v1.1

Herhangi bir verinin, e-posta bilgisi olup olmadığını kontrol etmek amacıyla kullanılır.

NoParametrelerAnlamları
1string E-posta AdresiKontrol edilecek veri.

echo isEmail('bilgi@zntr'); // Çıktı: false
echo isEmail('bilgi@zntr.net'); // Çıktı: true

Tadilat Ip Adreslerinden Biri Olup Olmadığını Kontrol Etmek

isRepmac() v1.0

Config/Repear.php dosyasında yer alan machines = array() dizisi içerisinde ip numarası veya numaralarının o anki modeminizin ip'si ile eşleşip eşleşmediğini kontrol eder. Böylece site içi tadilat yapılan bilgisayar ile diğer kullanıcı bilgisayarlarının ayırt edilmesi sağlanır.

echo isRepmac(); // Çıktı: true

Bir Verinin Metinsel veya Numerik Türe Uygunluğunu Kontrol Etmek

isChar(string $veri ) v1.2

Herhangi bir verinin metinsel veya numerik türde olup olmadığını kontrol etmek amacıyla kullanılır.

NoParametrelerAnlamları
1mixed VeriKontrol edilecek veri.

echo isChar('bilgi@zntr'); // Çıktı: true
echo isChar(3); // Çıktı: true
echo isChar(true); // Çıktı: false
echo isChar(array()); // Çıktı: false

Bir Verinin Hash Algoritmalarından Biri Olma Durumu Kontrol Etmek

isHash(string $algoritma ) v1.2

Parametreye girilen verinin hash algoritmalarından biri olup olmadığını kontrol edilir. Algoritmalardan biri ise true değeri döner. Aksi halde false değeri döner.

NoParametrelerAnlamları
1string VeriKontrol edilecek veri.

echo isHash('md5'); // Çıktı: true
echo isHash('xxx'); // Çıktı: false

Bir Verinin Encoding Karakter Setlerinden Biri Olma Durumu Kontrol Etmek

isCharset(string $karakter_seti ) v1.2

Parametreye girilen verinin tanımlı karakter setlerinden biri olup olmadığını kontrol edilir. Karakter setlerinden biri ise true değeri döner. Aksi halde false değeri döner.

NoParametrelerAnlamları
1string VeriKontrol edilecek veri.

echo isCharset('utf-8'); // Çıktı: true
echo isCharset('utf'); // Çıktı: false

Çerez Oluşturmak

Cookie::insert( string $cerez_ismi , string $cerezin_degeri , [ numeric $cerezin_suresi = 604800 ] , [ string $tarayici_dizini = '/' ] )

Çerez oluşturmak için kullanılır. 4 parametreden oluşur bunlar;

Opsiyonel KullanımNoParametrelerAnlamları
->name()1string Çerez İsmiOluşturulacak çerezin ad bilgisi.
->value()2string Çerezin DeğeriOluşturulan çerezin tutacağı değer.
->time()3[numeric Çerezin Süresi = 604800]Oluşturulan çerezin süresi. Varsayılan değeri 1 hafta.
->path()4[string Tarayıcı Dizini = '/']Çerezlerin tutulacağı dizin. Varsayılan değeri '/'.
->domain()5[string Domain]Çerezlerin geçerli olacağı domain bilgisi.
->secure()6[boolean Secure]Çerezler için https bağlantısının kullanılıp kullanılmayacağıdır.
->httpOnly()7[boolean Http Only]TRUE olduğu takdirde çerez sadece HTTP protokolü üzerinden erişilebilir olacaktır.

Çerezlerle ilgili ayarlar Config/Cookies.php dosyasında yer almaktadır. Config/ dizinindeki ayarları anlatırken bu ayarlar üzerinde duracağız. Şimdi çerez eklemeyi aşağıda bir örnek ile gösterelim.

<?php
class Cerezler
{
	public function index()
        { 
            Cookie::insert('cerez','Merhaba ZN'); // cerez adında bir çerez oluşturduk değerini de Merhaba ZN olarak belirledik.
        }
}

Şimdide oluşturduğumuz çerezi kullanalım.

Çerezleri Kullanmak

Cookie::select( string $cerez_ismi )

Oluşturulan çerezleri kullanmak için Cookie::select() yapısı kullanılır. Parametre olarak oluşturulan çerezin adı girilir.

Opsiyonel KullanımNoParametrelerAnlamları
->name()1string Çerez İsmiKullanılacak çerezin ad bilgisi.

<?php
class Cerezler
{
	public function index()
        {
            //Cookie::insert('cerez','Merhaba ZN'); 
            
            echo Cookie::select('cerez'); // Çıktı: 'Merhaba ZN'
            
        }
}

Çerezleri Silmek

Cookie::delete( string $cerez_ismi )

Oluşturulan çerezleri silmek için Cookie::delete() yapısı kullanılır. Parametre olarak oluşturulan çerezin adı girilir.

Opsiyonel KullanımNoParametrelerAnlamları
->name()1string Çerez İsmiSilinecek çerezin ad bilgisi.

<?php
class Cerezler
{
	public function index()
        {
            //Cookie::insert('cerez','Merhaba ZN'); 
            
            Cookie::delete('cerez'); // Olşuturulan cerez adındaki çerezi siliyoruz.
            
            echo Cookie::select('cerez'); // Herhangi bir çıktısı olmayacaktır çünkü üstte bu çerez silinmiştir.
            
        }
}

Çerezlerin Tümü Listelemek ve Silmek

Cookie::deleteAll() ve Cookie::selectAll()

Hangi çerezlerin oluşturulduğunu görmek için all_select() fonksiyonu kullanılırken tüm çerezleri silmek için all_delete() fonksiyonu kullanılmaktadır.

<?php
class Cerezler
{
	public function index()
        {
            //Cookie::insert('cerez','Merhaba ZN'); 
            
            var_dump(Cookie::selectAll()); // Olşuturulan çerezlerin listesini verecektir.
            
            Cookie::deleteAll(); // Olşuturulan tüm çerezleri siliyoruz.
            
        }
}

Çerezlerin Kullanımı Esnasında Hata Tespiti Yapmak

Cookie::error()

Çerezlerin kullanımı esnasında oluşan hataları tespit etmek için kullanılır. Hata yok ise false var ise hata değeri dönecektir.

echo Cookie::error(); // Hata var ise hata değeri döndürecektir.

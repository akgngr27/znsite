Encoding(Karakter Kodlamaları) Kullanmak

Bu bölümde veritabanı kütüphanesinde karakter kodlamaları ile alakalı yöntemlerden bahsedeceğiz.

Encoding(Karakter Kodlamaları)
Karakter Seti KullanmakDB::cset()
Karakter Grubu KullanmakDB::collate()
Karakter Seti ve Grubu KullanmakDB::encoding()

Karakter Seti Kullanmak

Oluşturulan tablonun veya sütunların karakter setini ayarlamak için kullanılır.

ParametreAnlamları
[1. Parametre = string Karakter Seti = utf8]Karakter seti adı.

    DB::addColumn('tablo', array('name' => array(DB::varchar(100), DB::cset('utf8'))));

Karakter Grubu Kullanmak

Oluşturulan tablonun veya sütunların karakter grubunu ayarlamak için kullanılır.

ParametreAnlamları
[1. Parametre = string Karakter Seti = utf8_general_ci]Karakter grubu adı.

    DB::addColumn('tablo', array('name' => array(DB::varchar(100), DB::cset('utf8'), DB::collate('utf8_general_ci'))));

Karakter Seti ve Grubu Kullanmak

Genellikle karakter seti ve grubu aynı anda kullırlar. Bu yöntemler yanyana kullanılacaklarında ikisini tek tek yazmak yerine encoding() yöntemi ile tek fonksiyonlar iki yöntemde kullanılabilir.

ParametreAnlamları
[1. Parametre = string Karakter Seti = utf8]Karakter seti adı.
[2. Parametre = string Karakter Seti = utf8_general_ci]Karakter grubu adı.

    DB::addColumn('tablo', array('name' => array(DB::varchar(100), DB::encoding('utf8', 'utf8_general_ci'))));

Iconv(IV) Kütüphanesi

Bu kütüphane PHP'nin iconv yöntemlerini kullanarak sınıf biçimine dönüştürülerek oluşturulmuştur. Yöntemlerin kullanımında parametre olarak farklılıklar vardır. İsimlendirmeleri akılda kalacak şekilde düzenlemeye özen gösterilmiştir.

Iconv(IV) işlevleri farklı dillerdeki karakter farklıklıklarından kaynaklı kodlama problemlerini çözmek için kullanılır.

Yöntemler arasında parametre kullanımında farklıklık olanlar tabloda belirtilmiştir.

Kullanım Farklılığı Olan Yöntemler

Kullanım farklı olan tek yöntem iconv() yöntemidir. Bu yöntemde dizge parametresi sonda yer almakta iken IV::convert() yönteminde 1. parametre olarak düzenlenmiştir. 

// Dizge sonra belirtiliyor.
iconv('UTF-8', 'ISO-8859-1//TRANSLIT', 'ZN FRAMEWORK');

// Önce dizge belirtiliyor.
IV::convert('ZN FRAMEWORK', 'UTF-8', 'ISO-8859-1//TRANSLIT');

iconv_get_encoding() ve iconv_set_encoding() yöntemlerinde input_encoding kullanımında _encoding eki IV:: yöntemlerinde kullanılmaz. 

// _encoding eki kullanılıyor.
iconv_get_encoding('input_encoding');

// _encoding eki kullanılmıyor.
IV::getEncoding('input');

Diğer yöntemlerin kullanımlarında hiç bir farklılık yoktur.

Data detaylı kullanım için iconv php.net kaynaklarını inceleyiniz.

Yönlendirme Adresini Belirlemek

Redirect::action( string $uri )

Yönlendirme adresi belirtilir. URL veya URI bilgisi girilebilir.

ParametreAnlamları
1. Parametre = URI BilgisiYönlendirme bilgisi.

Redirect::action('home/contact');

Yönlendirme Süresini Ayarlamak

Redirect::wait/time( numeric $sure )

Yönlendirme süresi belirtilir.

ParametreAnlamları
1. Parametre = Bekleme SüresiBekleme süresi bilgisi.

Redirect::wait(3)->action('home/contact'); // 3 saniye bekle ve yönlendir.

Yönlendirme ile Data Göndermek

Redirect::data/insert( array $data )

Yönlendirme ile gönderilecek datalar belirlenir.

ParametreAnlamları
1. Parametre = VerilerGönderilecek veriler.

Redirect::insert(array('data' => 'Data'))->wait(3)->action('home/contact'); // 3 saniye bekle ve yönlendir.

Yönlendirmeden Gelen Veriyi Kullanmak

Redirect::select( string $key )

Yönlendirme ile gelen veriyi kullanmak içindir.

ParametreAnlamları
1. Parametre = AnahtarKullanılmak istenen verinin anahtar değeri.

Redirect::select('data'); // Data.

Yönlendirmeden Gelen Veriyi Silmek

Redirect::delete( string/array $key )

Yönlendirme ile gelen veriyi silmek içindir.

ParametreAnlamları
1. Parametre = AnahtarKullanılmak istenen verinin anahtar değeri.

Redirect::delete('data');

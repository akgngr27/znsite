Sitenin Dil İçeriğini Ayarlamak

setLang( [ string $dil = 'tr' ] )

Sitenin dil içeriğini ayarlamak için kullanılır. Parametrenin varsayılan değeri 'tr', yani türkçedir. Sistemde varsayılan olarak iki dil ayarı mevcuttur. Bunlar; tr ve en 'dir. Bu ayarları değiştirmek yada daha fazla dil seçeneği eklemek için gerekli ayarlamaları Config/Language.php dosyasından yapmanız gerekmektedir. Unutmamanız gereken nokta ne kadar çok dil seçeneği eklerseniz Language/ dizini içerisindeki dil dizinlerinin sayısını da o denli artırmanız gerekmektedir. Yani Language/ dizinini kontrol ettiğinizde varsayılan olarak 2 adet dizin olduğunu görebilirsiniz. Bunlar; turkish ve english dizinleridir. Dikkat ederseniz dil dosyasının ayarları ile dil dizini içerisindeki dizinler arasında paralellik mevcuttur.

NoParametrelerAnlamları
1string Dil = "tr"Varsayılan olarak tr ayarlıdır.

echo getLang(); // Çıktı: tr
setLang('en');
echo getLang(); // Çıktı: en

Sitenin Dil İçeriğini Öğrenmek

getLang() ve currentLang() v1.0

Sitenin ayarlanmış dilini öğrenmek için kullanılır. Bu işlemi gerçekleştirmek için iki yöntem vardır. Bunlardan birincisi getLang() fonksiyonudur diğeri ise currentLang() fonksiyonudur. Esasen ikisinin de yaptığı iş aynıdır ancak aralarında bir fark vardır. Bu fark currentLang() yönteminin kullanılabilmesi için Config/Uri.php dosyasındaki $config['Uri']['lang'] = false; değerinin true olarak ayarlanması gerektiğidir. Bu ayar false olarak ayarlanırsa currentLang() yöntemi false değeri döndürür yani hiç bir çıktı vermez.

echo getLang(); // Çıktı: tr
setLang('en');
echo getLang(); // Çıktı: en
echo currentLang(); // Çıktı: false

Dil Dosyalarını Kullanmak

lang( string $dosya_adi, string $ayar_adi, string/array $degistirilecek_ifadeler ) v1.0

Sistemde tanımlı olan veya sizin oluşturacağınız dil dosyalarını kullanmak için bu yöntemden yararlanılır.

NoParametrelerAnlamları
1string Dosya AdıKullanılacak dil dosyasının adı.
2string Ayar AdıDil dosyasındaki kullanılacak ayarın adı.
3[ string/array Değiştirilecek Ayarlar]Veride yer alan karakterleri değiştirmek için kullanılır.

echo lang('Database', 'parameterError'); // Çıktı: % parametre eksik veya boş!
echo lang('Database', 'parameterError', 'TEST'); // Çıktı: TEST parametre eksik veya boş!
echo lang('Database', 'parameterError', array('%' => 'TEST', ...)); // Çıktı: TEST parametre eksik veya boş!

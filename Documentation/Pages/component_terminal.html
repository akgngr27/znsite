Terminal(Komut İşletim Sistemi) Kütüphanesi

Gerek PHP kodlarını çalıştırmak gerekse de Komut İşletim Sistemi komutlarını çalıştırmak için oluşturulmuştur. Kod üzerinden konsol ayarlarını yapılandırabileceğiniz gibi Config/Components.php ayar dosyasından da yapılandırmaları gerçekleştirebilirsiniz.

Table(Tablo) Kütüphanesini ve Yöntemleri
Terminal(Komut İşletim Sistemi) ÇalıştırmakTerminal::run()

Terminal(Komut İşletim Sistemi) Çalıştırmak

Terminal::run( [ string $tur = 'php' ] , array $ayarlar )

PHP ve komut işletim sistemi komutlarını çalıştırmak için kullanılır.

NoParametrelerAnlamları
1[string Tür = 'php']php, cmd parametrelerini alır.
Türler
phpPHP kodlarını çalıştırmak için bu ayar kullanılır.
cmdKomut işletim sistemi ayarlarını çalıştırmak için bu ayar kullanılır.
2[array Ayarlar]Terminalin görünümü ile ilgili ayarları içerir.
Ayarlar
widthTerminalin genişlik değeri.
heightTerminalin yükseklik değeri.
textColorYazı rengi.
textSizeYazı boyutu.
textTypeYazı türü.
bgColorArkaplan rengi.
barBgColorTemirnalin üst bölümünün arkaplan rengi.

Terminal::run('php', array('width' => 1000));

Komut işletim sistemi üzerinde yazılan komutları temizlemek için clear anahtar kelimesiniz kullanmanız yeterlidir.

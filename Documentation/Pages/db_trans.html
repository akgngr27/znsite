Transaction Sorgu Oluşturmak

Transaction sorgu oluşturmak için geliştirilmiştir. Transaction sorguların nasıl yapıldığı ile ilgili bilgiye bu yapıyla ilgili bilgi veren farklı kaynaklardan ulaşabilirsiniz.

Transaction Sogu Oluşturmak
Transaction Sorgusunu BaşlatmakDB::transStart()
Transaction Sorgusunu BitirmekDB::transEnd()
Transaction Sorgulara Örnekler

Transaction Sorgusunu Başlatmak

Transaction sorguya başlarken önce bu yöntem kullanılır.

DB::transStart();
// SORGULAR ...
// SORGULAR ...

Transaction Sorgusunu Başlatmak

DB::transStart()

Transaction sorguyu bitirmek için sorgular dizisinin en sonunda kullanılır.

DB::transStart();
// SORGULAR ...
// SORGULAR ...
 DB::transEnd();

Transaction Sorgulara Örnekler

DB::transStart()

Transaction sorguyu bitirmek için sorgular dizisinin en sonunda kullanılır.

DB::transStart();
DB::query('create table a');
DB::query('drop table ornektablo');
DB::transEnd();

DB::transStart();
DB::query('insert xx');
DB::query('drop database ornekveritabani');
DB::query('delete from ornektablo where id = 1');
DB::transEnd();

NOT
Transaction sorgular MYSQL yerine daha çok MYISAM, InnoDB veya BDB tablo türlerinde çalışır. Çoğu diğer veritabanı platformları transaction sorgu işlemlerini destekler.

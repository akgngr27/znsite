JQAjax İşlemini Tamamlama Fonksiyonu
Send JQAjax İşlemini TamamlamakJquery::ajax()->send()

Jquery Ajax İşlemleri

Jquery::ajax([$scriptOutput = false])->send( [ string $url ], [ string $data ] )

JQAjax kütüphanesi ile ilgili ajax veri gönderimine ait aşağıda örneklere yer verilmiştir. send() yöntemi gönderimi tamamlamak için kullanılan nihai yöntemdir.

echo Script::open(); // Script tagı açıyor ve script jquery dosyasını dahil ediyor.

echo Jquery::ajax()->url('test/ajax_islemleri') // Verinin gönderileceği URL belirtiliyor.
           ->data(JQ::serialize('#form') . ' + "data1=10&data2=20"') // Gönderilecek veriler ayarlanıyor.
           ->type('post') // Verilerin hangi yöntemle gönderileceği seçiliyor.
           ->dataType('json') // Veri türü ayarlanıyor.
           ->success('data', 'alert("Başarı ile gönderildi.");') // Verinin başarı ile gönderilip gönderilmediği kontrol ediliyor.
           ->error('data', 'alert("Hata oluştu!");') // Veri gönderiminde hata olup olmadığı kontrol ediliyor.
           ->send(); // Gönderim gerçekleştiriliyor.
           
echo Script::close() ;// Script tagını kapatıyor.

Eğer script taglarını fonksiyonun kendi içinde otomatik olarak yazmasını isterseniz ajax(true) olarak ayarlanmalıdır.

echo Jquery::ajax(true)->url('test/ajax_islemleri') // Verinin gönderileceği URL belirtiliyor.
           ->data(JQ::serialize('#form') . ' + "data1=10&data2=20"') // Gönderilecek veriler ayarlanıyor.
           ->type('post') // Verilerin hangi yöntemle gönderileceği seçiliyor.
           ->dataType('json') // Veri türü ayarlanıyor.
           ->success('data', 'alert("Başarı ile gönderildi.");') // Verinin başarı ile gönderilip gönderilmediği kontrol ediliyor.
           ->error('data', 'alert("Hata oluştu!");') // Veri gönderiminde hata olup olmadığı kontrol ediliyor.
           ->send(); // Gönderim gerçekleştiriliyor.

Çok kapsamlı bir gönderim yapmıyorsanız aşağıda gibi basit bir kullanım daha mevcuttur. 

echo Script::open(); // Script tagı açıyor ve script jquery dosyasını dahil ediyor.

echo Jquery::ajax()->send('test/ajax_islemleri', JQ::serialize('#form')); // 1. Parametre URL, 2. Parametre Data.
           
echo Script::close() ;// Script tagını kapatıyor.

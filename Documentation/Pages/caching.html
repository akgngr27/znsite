Cache(Ön Bellekleme) Kütüphanesi

Ön bellekleme oluşturma, kullanma ve silme gibi işlemeri gerçekleştiren sınıftır.

Cache(Ön Bellekleme) Kütüphanesi ve Yöntemleri

Ön Bellekleme Sürücüleri

Ön belleklemede kullanılabilir sürücüler aşağıdaki tabloda verilmiştir.

NoSürücülerAnlamları
1MemcacheKullanım için Memcache Kılavuz inceleyiniz.
2WincacheKullanım için Wincache Kılavuz inceleyiniz.
3ApcKullanım için Apc Kılavuz inceleyiniz.
4RedisKullanım için Redis Kılavuz inceleyiniz.
5FileDosya ön bellekleme için oluşturulumuştur.

Ön Bellekleme Oluşturmak

Cache::insert( string $nesne_ismi , variable $degisken , [ numeric $zaman = 1 ] , [ boolean $sikistirma = false ] )

Ön belleğe nesne almak için kullanılır. 4 parametreden oluşur bunlar;

NoParametrelerAnlamları
1string Nesne İsmiÖn belleğe alınacak nesnenin anahtar bilgisi.
2string Ön Belleğe Alınacak VeriÖn belleğe alınması istenilen veri bilgisi.
3[numeric Ön Bellekleme Süresi = 1]Ön belleğe alınma süresi.
4[boolean Sıkıştırma yapılsınmı = false]Sıkıştırma yapılıp yapılmayacağı bilgisi.

Cache::insert('nesne', $veri)

Farklı Sürücüler Kullanmak

Cache::driver( string $surucu_ismi )->...

Farklı bir sürücü kullanmak için bu yönteme parametre olarak geçerli ön bellekleme sürücülerinden birinin ismi yazılır.

NoParametrelerAnlamları
1string Sürücü İsmiKullanılabilir Sürücü İsimleri: memcache, wincache, apc ve file parametreleridir.

Cache::driver('apc')->insert('nesne', $veri);
Cache::driver('memcache')->insert('nesne', $veri);

Oluşturulan Ön Bellek Nesnesini Seçmek

Cache::select( string $nesne_ismi )

Ön belleğe nesne almak için kullanılır.

NoParametrelerAnlamları
1string Nesne İsmiÖn bellekten çağrılacak nesnenin anahtar bilgisi.

Cache::select('nesne')

Ön bellekten nesne silmek için kullanılır.

NoParametrelerAnlamları
1string Nesne İsmiÖn bellekten silinecek nesnenin anahtar bilgisi.

Cache::select('nesne')

Tüm Ön Belleği Boşaltmak

Cache::clean()

Tüm ön bellekteki verileri temizlemek için kullanılır.

Cache::clean()

Ön Bellekleme Hakkında Bilgi Almak

Cache::info()

Tüm ön bellekteki veriler hakkında bilgi almak için kullanılır.

Cache::info()

Ön Bellekleme Yapılan Nesne Hakkında Meta Bilgileri Almak

Cache::getMetaData( string $nesne_ismi )

Nesne hakkında bilgi almak için.

NoParametrelerAnlamları
1string Nesne İsmiÖn bellekten silinecek nesnenin anahtar bilgisi.

Cache::getMetaData('nesne')

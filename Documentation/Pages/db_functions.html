Fonksiyonların Kullanımı

Bu bölümde veritabanı fonksiyonlarının kullanımına yer verdik.

Fonksiyonların Kullanımı
Fonksiyonların KullanımıDB::func()

Fonksiyonların Kullanımı

DB::func( string $name , $column1, $column2, ... $columnN , [ $output = false ] )

DB::func() yöntemi aşağıda tabloda yer alan fonskiyonların işini yapmak için kullanılır. Ancak aşağıdaki fonksiyonlarında direk kullanımı mümkündür. Yani func() yöntemi alternatif olarak kullanılabilir.

ParametrelerKullanımları
1. Parametre Sütun adıHangi sütuna fonksiyon uygulanacaksa o sütunun adı belirtilir. Birden fazla sütunu yan yana yazmak sureti ile kullanabilirsiniz. Örnek: id, name
2. Parametre Değiştirilmiş sütun adıBu parametreye as ile başyan bir isim yazarsanız fonksiyonun uygulandığı sütunun değerini o isimle kullanabilirsiniz. Örnek: id, as userId. Eğer as kullanmaz iseniz sütun ismi olarak değerlendirilecektir.
[Son Parametre Çıktı:false]Bu parametre true olursa foksiyon string değer üretecektir.

    $row = DB::sum('id', 'as countId')->get('user')->row();
    echo $row->countId;
    

    $row = DB::sum('id', 'as countId')->max('total', 'as total')->get('user')->row();
    echo $row->countId;
    echo $row->total;

String veri çıktılatarak kullanımda mümkündür. Bunun için son parametrenin true olarak ayarlanması gerekmektedir.

    $row = DB::select('name', DB::sum('id', 'as countId', true), DB::max('total', 'as total', true))->get('user')->row();
    echo $row->countId;
    echo $row->total;
    echo $row->name;

Yukarıdaki yöntemlerin hepsini func() yöntemi ilede gerçekleştirebilirsiniz.

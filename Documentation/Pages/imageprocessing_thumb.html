Thumb(Resim Düzenleme) Kütüphanesi

Resimler üzerinde kesme, kırpma, küçültme gibi işlemler yapabilmek için oluşturulmuştur. Image kütüphanesinin kullanımı ile yaptığı işlem aynıdır. Sadece yöntemsel açıdan farklılıklar vardır.

Thumb Kütüphanesi Yöntemleri

Thumb kütüphanesine ait yöntemler aşağıdaki tabloda listelenmiştir.

Thumb Oluşturma YöntemleriAnlamları
path( $path ) Hangi resim üzerinde düzenleme yapılacaksa onun yolu belirtilir. Örnek: ->path('Application/Resources/Uploads/resim.jpg')
quality( $quality ) Düzenleme yapılan resmin kalitesidir. Örnek: ->quality(80)
crop( $x, $y ) Resmin hangi koordinatlar kırpılmaya başlayacağı bilgisidir. X yatayda, Y ise dikeyde kaçıncı pikselden itibaren kırpılmaya başlanacağını belirtir. Örnek: ->crop(100, 100)
size( $width, $height ) Resmin boyutunu küçültmek için kullanılır. Örnek: ->size(400, 200)
resize( $width, $height ) Resmi yeniden boyutlandırmak için kullanılır. Bu yöntemin size() yönteminden farkı boyutlarında da küçülme olmasıdır. Örnek: ->resize(400, 200)
prosize( $width, $height ) Resimde ortantılı boyutlandırma yapmak için kullanılır. Bu yöntemde genelde tek parametre kullanılır. Diğer parametre bu parametrenin oranına göre kendini ayarlar. Örnek: ->prosize(400)

Yukarıdaki yöntemlerin kullanımına yönelik örnekler aşağıda verilmiştir.

Thumb::path('resim.jpg')  // İşlem görecek resmin yolu belirtiliyor.
     ->crop(100, 200)  // Kaçıncı pikselden kırpılmaya başlanacağı belirtiliyor.
     ->resize(300, 200)  // Genişliği 300, yüksekliği 200 olarak yeniden boyutlandırılıyor.
     ->create();

Resmin Orantılı Ölçüsünü Almak

Bu yöntem sayesinde yolu belirtilen bir resmin uzunluk değerlerinden birine göre diğerinin orantılı ölçüsü alınabilir.

Thumb Oluşturma YöntemleriAnlamları
path( $path ) Resmin yolunu belirtmek için kullanılır. Örnek: ->path('Application/Resources/Uploads/resim.jpg')
getProsize( $width, $height ) Resmin genişlik veya yükseklik değerlerinden biri kullanılarak diğer uzunluğu orjinal resimdeki uzunluk oranlarına göre orantısı alınır. Örnek: ->getProsize(80) // Genişliği 80 piksel olarak belirnen nesnenin yükseklik ölçüşü öğrenilmek isteniyor. Çıktısı bir object veri türünde değer döndürür. 

 // Resmin boyutları 1000 x 500 olsun.
$resim = Thumb::path('resim.jpg')  // Ölçüsü alınacak resmin yolu belirtiliyor.
              ->getProsize(100);  // Resmin orantılı boyutları alınıyor.
                    
echo $resim->height;   // 50 Genişlik değerine göre orantılanmış yükseklik ölçüsüdür.

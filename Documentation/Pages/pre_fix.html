Bir Veriye Önek Eklemek

prefix( string $veri , [ string $onek = '/' ] )

Herhangi bir metinsel veriye ön ek eklemek için kullanılır.

NoParametrelerAnlamları
1string VeriÖn ek eklenecek veri.
2[ string Önek = "/" ]Ön ek.

echo prefix('Veri'); // Çıktı: /Veri
echo prefix('Veri', '+++'); // Çıktı: +++Veri

Eğer eklenecek ön ek, verinin önünde zaten varsa ayrıca ekleme işlemi yapmaz.

echo prefix('/Veri'); // Çıktı: /Veri
echo prefix('+++Veri', '+++'); // Çıktı: +++Veri

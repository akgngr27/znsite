Veri Temizlemek

Clean::data( string/array $diziVeyaMetin , string/array $temizlenecekIfade )

Dizilerde veya metinlerde ifade temizlemek için kullanılır.

NoParametrelerAnlamları
1string/array Dizi veya MetinHangi dizi veya metinde silme işlemi yapılacağı.
2string/array Temizlenecek İfadeHangi ifade veya ifadelerin silineceği.

/* --- Metinsel ifadelerde temizleme işlemi --- */
echo Clean::data('bilgi@zntr.net', 'bilgi'); /* Çıktı: @zntr.net */
echo Clean::data('bilgi@zntr.net', array('bilgi', '.net')); /* Çıktı: @zntr */

/* --- Dizi İçerikli ifadelerde temizleme işlemi --- */
output( Clean::data(array('a', 'b', 'c'), 'b') ); /* Çıktı: a c */
output( Clean::data(array('a', 'b', 'c'), array('b', 'c')) ); /* Çıktı: a */

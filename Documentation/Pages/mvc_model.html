Model Kullanımı

Models/ dizini, içerisinde oluşturulacak olan model dosyaları kullanımının temel mantığı veritabanı sorgularının bir kez yazılıp tekrar tekrar kullanılmasını sağlamaya dayanır. Sadece sorgular mı yazılır? Hayır istenirse harici olarak dahil edilmesi düşünülen farklı kodlarda yazılabilir.

Model Dosyası Oluşturun

Aşağıdaki gibi örnek bir model dosyası oluşturabilirsiniz.

<?php
class OrnekModel extends Controller // ya da Model
{
    public function get()
    {
         return $this->db->get('OrnekTablo')->result();
    }
}

Yukarıdaki kullanım sadece örnek amaçlıdır.

Oluşturulan Model Dosyasını Kullanın

Model dosyası hangi sınıf ismi ile oluşturulmuşsa o isimle kullanılır.

$this->ornekmodel->get();

Models Dosyalarında Namespace Kullanmak

Eğer model dosyalarınız içinde namespace tanımlaması yapmışsanız aşağıdaki gibi bir kullanım gereklidir.

Models\Ornek\OrnekModel.php 
<?php

namespace Ornek;
use Controller;

class OrnekModel extends Controller // ya da Model
{
    public function get()
    {
         return $this->db->get('OrnekTablo')->result();
    }
}

Şimdi bu model dosyasını kullanmak için;

$this->{'Ornek\OrnekModel'}->get();

Ya da.

$this->OrnekModel->get();

Şeklinde kullanabilirsiniz.

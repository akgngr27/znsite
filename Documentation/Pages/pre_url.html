Sayfanın Aktif URL'sini Öğrenmek

currentUrl()

Herhangi bir web sayfasının o an aktif olan URL adresini öğrenmek için kullanılır.

echo currentUrl(); // Çıktı: http://www.siteadi.xxx/ornek/uygulama

Sitenin URL Adresini Öğrenmek

siteUrl( [ string $url_eki ] , [ numeric $dizin_indeksi = 0 ] ) v1.0

Sitenin URL adresini öğrenmek için kullanılır. Config/Uri.php dosyasından ayarlar yapılırsa bu değişikliklerde dikkate alınarak sitenin ayarlanmış URL adresini verir.

NoParametrelerAnlamları
1string URL EkiSite URL adresine ek olarak yazılacak URI ifadeleri.
2[ string Dizin Indeksi ]URL'nin üst dizinlerine ulaşabilmek için kullanılır.

echo siteUrl(); // Çıktı: http://www.siteadi.xxx/zntr/index.php/
echo siteUrl('ornek/uygulama'); // Çıktı: http://www.siteadi.xxx/zntr/index.php/ornek/uygulama/
echo siteUrl('ornek/uygulama', -1); // Çıktı:http://www.siteadi.xxx/index.php/ornek/

Eğer Config/Uri.php dosyasından ssl, dil ve index.php ayarlarıda yapılmış olursa bu değişikliklerde dikkate alınır.

echo siteUrl(); // Çıktı: https://www.siteadi.xxx/tr/zntr/
echo siteUrl('ornek/uygulama'); // Çıktı: Çıktı: https://www.siteadi.xxx/tr/zntr/ornek/uygulama

Sitenin Kök URL Adresini Öğrenmek

baseUrl( [ string $url_eki ] , [ numeric $dizin_indeksi = 0 ] ) v1.0

Sitenin Kök URL adresini öğrenmek için kullanılır. Config/Uri.php dosyasındaki ayarlardan etkilenmez. Genellikle bu yöntem gerçek dosyaları kaynak olarak göstermek için kullanılır. Özellikle resim, ses veya vidyo dosyalarının yolu belirtilmek istendiğinde tercih edilir.

NoParametrelerAnlamları
1string URL EkiSite URL adresine ek olarak yazılacak URI ifadeleri.
2[ string Dizin Indeksi ]URL'nin üst dizinlerine ulaşabilmek için kullanılır.

echo baseUrl(); // Çıktı: http://www.siteadi.xxx/zntr/
echo baseUrl('ornek/uygulama/resim.jpg'); // Çıktı: Çıktı: http://www.siteadi.xxx/zntr/ornek/uygulama/resim.jpg
echo baseUrl('ornek/uygulama/resim.jpg', -1); // Çıktı: Çıktı: http://www.siteadi.xxx/ornek/uygulama/resim.jpg

Config/Uri.php dosyasında yer alan dil ve index.php ayarlarındaki değişiklik baseUrl() yönteminde herhangi bir değişikliğe neden olmaz.

echo baseUrl(); // Çıktı: https://www.siteadi.xxx/zntr/
echo baseUrl('ornek/uygulama/resim.jpg'); // Çıktı: Çıktı: https://www.siteadi.xxx/zntr/ornek/uygulama/resim.jpg

Gelinen Bir Önceki URL Adresini Öğrenmek

prevUrl() v1.0

Ziyaret edilen sayfadan bir önceki sayfanın URL adresini öğrenmek için kullanılır.

echo prevUrl(); // Çıktı: http://www.siteadi.xxx/zntr/urunler/

Host URL Adresini Öğrenmek

hostUrl( [ string $url_ekli ] ) v1.0

Sistemin kurulu olduğu hostun URL adresini verir.

NoParametrelerAnlamları
2[ string URL Eki = "" ]Host URL adresinin sonuna eklenecek ifadeler.

echo hostUrl(); // Çıktı: http://www.hostadi.xxx/
echo hostUrl('ornek/uygulama'); // Çıktı: http://www.hostadi.xxx/ornek/uygulama

Host Bilgisini Öğrenmek

host() v1.2

Sistemin host bilgisini öğrenmek için kullanılır.

echo host(); // Çıktı: localhost
echo host(); // Çıktı: localhost:8080
echo host(); // Çıktı: www.ornek.xxx

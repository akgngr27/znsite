Dizilerde ve Nesnelerde Düzenli Çıktı Oluşturmak

output(array/object $var) v1.4

var_dump gibi kullanılan bu yöntem daha düzenli görsel çıktı oluşturur.

NoParametrelerAnlamları
1array Çıktısı İstenilen DiziDüzenli çıktısı istenilen dizi veya object nesnesi.

output(array(1, 2)); // Çıktı: Düzenli dizi içeriği.

Çıktı Oluşturmak

write(mixed $data, array $vars) v1.4

Ekranda çıktı oluşturmak için kullanılır. 2. parametre ile çıktıya veri gönderilebilir.

NoParametrelerAnlamları
1mixed VeriÇıktısı istenilen veri.
2array DeğişkenlerÇıktısı istenilen veriye değer göndermek için kullanılır.

write('Merhaba {0} {1}', array('Dünyaya', 'Hoş Geldiniz!')); // Çıktı: Merhaba Dünyaya Hoş Geldiniz.
write('Merhaba {x} {y}', array('x' => 'Dünyaya', 'y' => 'Hoş Geldiniz!')); // Çıktı: Merhaba dünyaya hoş geldiniz.

Çıktı Oluşturmak

writeLine(mixed $data, array $vars, [ numeric $br_count = 1 ]) v1.4

Ekranda çıktı oluşturmak için kullanılır ancak bu yöntemin write() yönteminden farkı çıktıdan sonra bir alt satıra geçilmesidir. 2. parametre ile çıktıya veri gönderilebilir.

NoParametrelerAnlamları
1mixed VeriÇıktısı istenilen veri.
2array DeğişkenlerÇıktısı istenilen veriye değer göndermek için kullanılır.
3[ numeric Alt Satır Sayısı = 1 ]Çıktıdan sonra kaç satır alta geçileceğini belirtmek için kullanılır. Varsayılan:1

writeLine('Merhaba {0} {1}', array('dünyaya', 'hoş geldiniz!'));
writeLine('Merhaba {0} {1}', array('Hayat', 'çok güzel!'));
// Merhaba dünyaya hoş geldiniz.
// Merhaba hayat çok güzel.

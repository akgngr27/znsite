E-posta Ayarlarını Yapılandırmak

Config/Email.php dosyasın yer alan varsayılan ayarlar aşağıdaki gibidir.

Ayarlar Varsayılan Seçenekler Açıklamalar
driver mail mail, smtp, pipe, imap, send E-posta gönderim türü.
mailPath /usr/sbin/sendmail - E-posta sunucu yolu
smtpHost - - SMTP sunucu adresi.
smtpUser - - SMTP kullanıcı e-posta adresi.
smtpPassword - - SMTP kullanıcı şifresi.
smtpPort 587 - SMTP port numarası.
smtpTimeout 10 - SMTP zaman aşımı süresi (saniye).
smtpKeepAlive false true, false Kalıcı SMTP bağlantılarını etkinleştirin.
smtpEncode tls tls, ssl SMTP şifreleme yöntemi
senderMail - - From işlevi için ön tanımlı gönderen e-posta adresi.
senderName - - From işlevi için ön tanımlı gönderen ismi.
contentType text text, html E-posta içeriğinin html mi yoksa text içeriklik olup olmayacağının belirlenmesi içindir.
charset utf-8 - Karakter seti (utf-8, iso-8859-1, etc.).
priority 3 1, 2, 3, 4, 5 Email Priority. 1 = En yüksek. 5 = En düşük. 3 = normal.
dsn false true, false Sunucudan iletiyi bildirmeyi etkinleştirme.
xMailer ZN - X-Mailer ve User-Agent başlıkları için kullanılan ayar.

E-posta Göndermek

Basit e-posta gönderimi için aşağıda örnekler verilmiştir.

Email::from('bilgi@zntr.net', 'ZNTR')  // veya sender()
     ->to('ornek@ornek.com')  // veya receiver()
     ->cc('ornek2@ornek.com')
     ->bcc('ornek3@ornek.com')
     ->subject('Konu')
     ->message('Mesaj')  // veya content()
     ->send();
     
output(Email::error());

E-posta Ayarlarını Yapılandırmak

Ayarlar Config/Email.php dosyasından yapılabileceği gibi gönderim yapılırken settings() yöntemi ilede yapılabilir.

Email::settings(array('senderName' => 'ZNTR', 'senderMail' => 'bilgi@zntr.net'))
      // ->sender('bilgi@zntr.net', 'ZNTR')
     ->receiver('ornek@ornek.com')
     ->subject('Konu')
     ->message('Mesaj')
     ->send();
     
output(Email::error());

Gönderici Bilgisi Oluşturmak

->from( string $eposta , string $isim , [ string $donusEposta ] )

->sender( string $eposta , string $isim , [ string $donusEposta ] )

E-posta gönderiminde göndericinin e-posta ve isim bilgisini oluşturmak için kullanılır.

NoParametrelerAnlamları
1string E-posta AdresiGönderici e-posta adresi.
2string İsimİndirilecek dosyanın yol bilgisi.
3[string Dönüş E-posta Adresi]İsteğe bağlı e-posta adresi ile teslim edilmemiş e-posta yönlendirmek için kullanılır.

Email::sender('bilgi@zntr.net', 'ZNTR');

Email::from('bilgi@zntr.net', 'ZNTR', 'donus@epostaadresi.com');

E-posta İçerisiden Dosya Göndermek

->attachment( string $dosyaAdi , string $egilim , string $yeniIsim , string $mimeTipi)

E-posta metninin yanında dosyada göndermek için kullanılır.

NoParametrelerAnlamları
1string Dosya AdıGönderilecek olan dosyanın adı yani yolu.
2string EğilimVarsaylan eğilim belirtilir. Geçerli bir eğilim belirtilemiyorsa boş bırakılmalıdır.
Geçerli Eğilim İsimlendirmeleriAnlamları
inlineOtomatik olarak görüntülenir.
attachmentKullanıcı kontrollü gösterilir.
form-dataForm cevap olarak işlem görülür.
signalTünelli içerik sessizce işlenir.
alertMesaj kullanıcıyı uyarmak için özel bir uyarı kullanır.
iconMesaj kullanıcıya bir simge olarak görüntülenir.
renderMesaj kullanıcıya görüntülenmesi gereken gibi görüntülenir.
recipient-list-historyMesaj isteği, alıcılarını gösteren URI listesini içerir.
sessionMesaj RFC2327 SDP formatın bir oturum tarif edilir.
aibDoğrulanmış kimlik.
early-sessionMesaj RFC2327 SDP formatın bir oturum tarif edilir.
recipient-listMesaj URI listesi hizmetleri uygulayacak herhangi bir URI listesini içerir.
notificationMesaj anlık ileti Bırakma bildirimi olarak görüntülenir.
by-referenceMesaj aynı SIP mesajında yer alan içeriğe göre ele alınmasını gerektirir.
info-packageVücudun bir bilgi paketi ile ilgili bilgileri içerir.
3string Yeni İsimDosyanın görüntülenecek yeni ismi.
4string Mime TipiGeçerli mime tiplerinden biri kullanılır. Config/MimeTypes.php dosyasında kullanılabilir mimi tipleri yer almaktadır.

Email::attachment('dosya/resim1.jpg');
Email::attachment('dosya/resim2.jpg', 'inline', 'yeniResim.jpg');

Email::attachment('dosya/resim2.jpg', '', 'yeniResim.jpg', 'image/jpeg');

E-posta Dosyasının İçeriği Hakkında Bilgi Almak

->attachmentContentId( string $dosyaAdi )

E-posta gönderiminde gönderilecek dosya hakkında içerik bilgisi almak için kullanılır.

NoParametrelerAnlamları
1string Dosya Adıİçerik bilgisi alınacak dosyanın adı.

$dosyaAdi = 'dosya/resim1.jpg';
Email::attachment($dosyaAdi);

foreach($epostaListesi as $eposta)
{
        Email::to($eposta);
        $icerikId = Email::attachmentContentId($dosyaAdi);
        Email::message('<img src='contentId:". $icerikId ."' alt="resim1" />');
        Email::send();
}

E-posta Gönderiminde Hata Bilgisi Almak

Email::error()

E-posta gönderiminde hata bilgisi almak için Error kütüphanesinden yararlınılır.

output(Email::error());

Diğer E-posta Yöntemleri

Bu bölümde e-posta kütüphanesinın diğer yöntemlerinin nasıl kullanıldığı ile ilgili anlatıma yer verdik.

Diğer Yöntemler
YöntemİşlevKullanım
smtpHost() SMTP sunusunu ayarlamak için kullanılır. // @sunucuAdi string => Sunucu ismi belirtmek içindir.
 Email::smtpHost('mail.sunucuadi.com');
// ... empty ...
smtpUser() SMTP kullanıcı adını ayarlamak için kullanılır. // @kullaniciAdi string => SMTP kullanici ismi.
 Email::smtpUser('test@test.com');
// ... empty ...
smtpPassword() SMTP kullanıcı şifresini ayarlamak için kullanılır. // @kullaniciSifresi string => SMTP kullanici sifresini.
 Email::smtpPassword('test1234');
// ... empty ...
smtpPort() SMTP bağlantı portunu ayarlamak için kullanılır. // @port string => SMTP port bilgisi.
 Email::smtpPassword('test1234');
// ... empty ...
smtpTimetout() SMTP bağlantı zaman aşımı süresini ayarlamak için kullanılır. // @zamanAsimi numeric => SMTP zaman aşımı bilgisi.
 Email::smtpTimetout(10);
// ... 10 ...
smtpKeepAlive() SMTP bağlantısı açık tutulsun mu?. // @baglanti boolean => True veya false.
 Email::smtpKeepAlive(true);
// ... true ...
smtpEncode() SMTP şifreleme yöntemi belirtilir. Tls veya ssl // @tur string => tls.
 Email::smtpEncode('ssl');
// ... tls ...
mailPath() Sunucu e-posta dizini yolunu belirtmek için kullanılır // @tur string => 'usr/sbin/sendmail'.
 Email::mailPath('usr/sbin/sendmail');
// ... 'usr/sbin/sendmail' ...
dsn() E-posta teslim durum bilgisi kullanılsın mı?. // @status boolean => false.
 Email::dsn(true);
// ... false ...
charset() E-posta karakter setini ayarlamak içindir. // @status string => 'utf-8'
 Email::charset('utf-8');
// ... utf-8 ...
multiPart() E-posta çoklu bölüm kullanımı içindir. // @multi string => 'mixed'
 Email::multiPart('related');
// ... mixed ...
contentType() E-posta içerik türünü belirlemek için kullanılır. text veya html. // @tur string => text
 Email::contentType('html');
// ... text ...
priority() E-posta gönderim önceliğini belirlemek için kullanılır. 1, 2, 3, 4, 5. // @tur numeric => 3
 Email::priority(3);
// ... 3 ...

Template(Şablon) Kütüphanesi

Bu kütüphanenin verileri manüpüle etmek için bir data() yöntemi bulunsada genel olarak kullanımı Views/ dizindeki görünüm sayfalarının sonuna .template uzantısı getirilerek kullanımıdır. Template kütüphanesinin kurallarına uygun yazacağınız her görünüm sayfası bu kütüphane ile ayrıştırılarak normal php kodlarına çevrilir. Şimdi bu kütüphaneyi nasıl kullanarız onu gösterelim.

Öncelikle yapmanız gereken Views/test.template.php örnek bir görünüm dosyası oluşturmak.

Şablon Görünüm Dosyasında Fonksiyon Kullanmak

Şablon görünüm sayfasında aşağıdaki şekilde fonksiyonlar kullanılabilir.

KullanımAnlamları
@fonksiyonEkrana bastırılmayacak fonksiyonlar için kullanılır.
@@fonksiyonEkrana bastırılacak fonksiyonlar için kullanılır.

Viewst/test.template.php 
@print('test') // Sonuc: test

Ekrana çıktı vermesini istediğimiz foksiyonları ise aşağıdaki gibi kullanıyoruz.

Viewst/test.template.php 
@@siteUrl('test') // Sonuc: http://site.com/test

Şablon Görünüm Dosyasında Karar Yapıları Kullanmak

Şablon görünüm sayfasında aşağıdaki şekilde karar yapıları kullanılabilir.

KullanımAnlamları
@if(), @elseif(), @else, @endifİf - else kullanımı.

If-Else Kullanımı

Viewst/test.template.php 
@if( $deger === 1 )
    {{ 1 }}
@elseif( $deger === 'test' )
    {{ 'test' }}
@else
    @print('varsayılan')
@endif

Şablon Görünüm Dosyasında Döngü Kullanmak

Şablon görünüm sayfasında aşağıdaki şekilde döndüler kullanılabilir.

KullanımAnlamları
@for(), @endforFor döngüsünün kullanımı.
@foreach(), @endforeachForeach döngüsünün kullanımı.
@while(), @endwhileWhile döngüsünün kullanımı.

For Kullanımı

Viewst/test.template.php 
@for($i = 0; $i < 10; $i++)
	@writeLine($i)
@endfor

Diğer döngülerin kullanımıda benzerdir.

Açıklama Satırı Kullanmak

Şablon görünüm sayfasında aşağıdaki şekilde açıklama kullanılabilir.

KullanımAnlamları
{-- Açıklama Satırı --}Açıklama satırı kullanımı.

Viewst/test.template.php 
@for($i = 0; $i < 10; $i++)
	@writeLine($i) {-- Bu bölüm açıklama satırıdır. --}
@endfor

Yazdırma Tagları Kullanmak

Şablon görünüm sayfasında aşağıdaki şekilde yazdırma işlemi uygulanabilir.

KullanımAnlamları
{{ 'Yazdırılacak Metin' }}İfade dizge ise tırnak içinde kullanılmalıdır.

Viewst/test.template.php 
{{ 'Merhaba ZERONEED' }}

PHP Tagları Kullanmak

Şablon görünüm sayfasında aşağıdaki şekilde php tagları kullanılabilir.

KullanımAnlamları
{[ ]}Bir süslü ve köşeli parantez ile.

Viewst/test.template.php 
{[ echo 'Merhaba ZERONEED' ]}

Yazdırılabilir Verileri Kullanmak

Controllers/ kontrolcü sayfalarından gönderilen dataları kullanmak için { data } formundan yararlanırız. Yazdırılabilir verilerden kasıt ölçülebilir büyüklükler(string, boolean, integer) kastedilmektedir.

KullanımAnlamları
{ data }Yazdırılabilir veri.
$dataVerimiz yazdırılabilir bir veri değilse bu şekilde de bir kullanımda mümkündür.

Controllers/test.php kontrolcüsünden test datasının gönderildiğini varsayarsak bu datayı aşağıdaki gibi kullanabiliriz.

Viewst/test.template.php 
{ test }

Yazdırılabilir Olsun Olmasın Tüm Verileri Kullanmak

Controllers/ kontrolcü sayfalarından gönderilen dataları kullanmak için $data formundan yararlanırız.

KullanımAnlamları
$dataGelen verileri değişken olarak kullanmak mümkündür.

Controllers/test.php kontrolcüsünden test datasının gönderildiğini varsayarsak bu datayı aşağıdaki gibi kullanabiliriz.

Viewst/test.template.php 
{{ $test}}

Aşağıdaki gibi kısa yoldan değişkeni yazdırmakta mümkündür.

Viewst/test.template.php 
@$test

Ayrıştırıcı Grubu Oluşturmak

Ayrıştırıcı grubu oluşturmak için aşağıdaki yöntemi kullanabilirsiniz.

Views/test.template.php 
<ul>
    { elements }
        <li><a href="{ href }">{ name }</a></li>
    { / elements }
</ul>

Controllers/test.php 
$data = array
        (
            'elements' => array
            (	
            	array('href' => 'Link1', 'name' => 'LinkName1'),
                array('href' => 'Link2', 'name' => 'LinkName2'),
                array('href' => 'Link3', 'name' => 'LinkName3')
            )
        );
        
Import::view('test.template', $data);

// Çıktı: <ul> <li><a href="Link1">LinkName1</a></li> <li><a href="Link2">LinkName2</a></li> <li><a href="Link3">LinkName3</a></li> </ul>

Özel HTML Yapısı Kullanmak

İsteğe bağlı olarak aşağıdaki tipte html etiketlerini kullanmanızda mümkündür.

KullanımAnlamları
#tag[ozellik="deger"]Tag açmak için # ifadesini kullanmanız yeterlidir.
#endtag, ##tagTagı kapatmak için end veya fazladan 2 # kullanmanız yeterlidir.
#tag(İçerik, ozellik="deger")Tagı açar içeriği ekler ve kapatır. 2 parametre ile isternise özellik değer çifti eklenebilir.
##tag(ozellik="deger")Tagı kapatılmaya ihtiyaç duymayan etiketler içindir. Örnek br etiketi.

Viewst/test.template.php 
#p
	Bu bölüm #u paragraf ##u bölümüdür #br
##p

Ya da end ibaresi ile taglarınızı kapatabilirsiniz.

Viewst/test.template.php 
#p
	Bu bölüm #u paragraf #endu bölümüdür #br
#endp

# karakteri normal olarak kullanılmak istenirse herhangi bir alfabetik harf ile birleştirilmeden kullanılabilir.

Viewst/test.template.php 
#u[type="disc"]
    #li # Madde 1 ##li
    #li # Madde 2 ##li
##u

#tag() Kullanımı.

Viewst/test.template.php 
#title(Başlık Bilgisi) // <title>Başlık Bilgisi</title>

##tag() Kullanımı.

Viewst/test.template.php 
##input(type="submit" value="Buton") // <input type="submit" value="Buton">

Kelime Sayısını Sınırlamak

Limit::word( string $metin , [ numeric $limit = 100 ] , [ string $karakter_sonu_eki = '...' ] , [ boolean $html_taglarini_temizle = true ] )

Girilen metindeki limitlenecek kelime miktarı belirlenir.

NoParametrelerAnlamları
1string MetinLimitleme uygulanacak metin.
2[ string/int limit = 100 ]Limitlenecek kelime miktarı.
3[ string Karakter Sonu = "..." ]Limitleme işlemi bittiğinde metnin sonuna gelecek devam eki.
4[ boolean = true ]Html tagları temizlensin mi?.

echo Limit::word("Merhaba Zntr'ye hoş geldiniz", 2); // Çıktı: Merhaba Zntr'ye...
echo Limit::word("Merhaba Zntr'ye hoş geldiniz", 3, '+++'); // Çıktı: Merhaba Zntr'ye hoş+++

Style Kütüphanesi

Stil tagı açmak kapamak veya harici stil yüklemek üzerine geliştirilmiş kütüphanedir.

Style Kütüphanesi Yöntemleri

Style kütüphanesine ait yöntemler aşağıdaki tabloda listelenmiştir

Style Kütüphanesi ve YöntemleriAnlamlarıKullanımları
type( $tip = 'text/css' )Varsayılan olarak ayarlıdır. Gerekmedikçe kullanım gerektirmez.->type('text/css')
library( $eklenecekKutuphaneler )Harici css dosyaları dahil edilmesi düşünülüyorsa bu yöntemin içerisine parametre olarak belirtilebilir.->library('Style', 'Anasayfa')
open()Style tagı açmak işlevini üstlenir.->open();
close()Style tagı kapatma işlevini üstlenir.->close();

Yukarıdaki yöntemlerin kullanımına yönelik örnekler aşağıda verilmiştir.

Style::open();
	// stil kodları 
Style::close();

Style tagı açarken farklı harici css dosyası dahil etmek isterseniz.

Style::library('style1', 'style2', 'style3')->open();
  // stil kodları 
Style::close();

JQAnimate Kütüphanesi

Jquery'nin animasyon komutları kullanılarak oluşturulmuştur. Daha basit bir şekilde hareket işlemlerini gerçekleştirebilmeniz mümkündür.

Zincirleme Kullanımlı JQAnimate Parametreleri
Seçiçi BelirlemekJquery::animate()->selector(string $selector)
Özellikler EklemekJquery::animate()->attr(array $properties, string $code)
Hızı AyarlamakJquery::animate()->duration/speed(string/numeric $speed)
Kuyruğu Kontrol EtmekJquery::animate()->queue(string/bool $queue)
Easing Efekti UygulamakJquery::animate()->easing(string $type)
Özel Efektler KullanmakJquery::animate()->specialEasing(array $easings)
Geri Dönüş Fonksiyonu KullanmakJquery::animate()->callback/func(string $parameter, string $code)
Step Fonksiyonu KullanmakJquery::animate()->step(string $code)
Complete Fonksiyonu KullanmakJquery::animate()->comp(string $code)
Hareketi Kuyruğa Ekleyecek Şekilde TamamlamakJquery::animate()->complete()
Hareketi OluşturmakJquery::animate()->create(arguments $complete1, $complete2 ... $completeN)

Zincirleme Kullanımlı Jquery Hareket İşlemleri

Jquery::animate([$scriptOutput = false])->create( [ arguments $complete1, $complete2 ... $completeN ] )

JQAnimate kütüphanesi ile ilgili animasyonun nasıl yapılacağına ait aşağıda örneklere yer verilmiştir. create() yöntemi gönderimi tamamlamak için kullanılan nihai yöntemdir.

Aşağıda animasyonun nasıl oluşturulacağına dair bir örneğe yer verilmiştir.

echo Script::open(); // Script tagı açıyor ve script jquery dosyasını dahil ediyor.

echo Jquery::animate()->selector('#button')
              ->attr(array('width' => 1000, 'height' => 200))
              ->speed(1000)
              ->easing('linear')
              ->callback('e', 'alert("Animasyon Tamamlandı.")')
              ->create();
           
echo Script::close();// Script tagını kapatıyor.

// Kaynak Kod: $("#button").animate({width:1000, height:200},{duration:1000, easing:"linear", complete:function(){alert("Animasyon Tamamlandı.");}});

Harekete kuyruk eklemek için complete() yöntemini aşağıdaki gibi düzenlemeniz yeterlidir.

->create(JQ::property('show', 1000), JQ::property('fadeIn', array('slow', 'linear')));
// Kaynak Kod: 
$("#button").animate
({
    width:1000, height:200},{duration:1000, easing:"linear", complete:function(){alert(1);}
}).hide(1000).fadeIn("slow","linear");

Http Kütüphanesi

Method kütüphanesi kullanılarak $_GET, $_POST, $_EVN, $_SERVER, $_REQUEST süper globallerini kullanmak için oluşturulmuştur. Bu kütüphane Method:: kütüphanesini kullandığı gibi Config/Http.php ayar dosyasında yer alan verileride kullanmaktadır. İşlev olarak method kütüphanesi ile aynı işi yapıyor olsada işlevlerin kullanımında ufak tefek farklılıklar vardır.

Network(Net) Kütüphanesi ve Yöntemleri
Http Yöntemlerini KullanmakHttp::input/name/value/select/insert/delete()
Config/Http.php Dosyasında Yer Alan Verileri KullanmakHttp::code/message()
Ajax Verisi Kontrolünü YapmakHttp::isAjax()

Http Yöntemlerini Kullanmak

Http::input( string $type )

Hangi süper globalin kullanılacağı seçilir.

ParametreAnlamları
1 - string Girdi TürüHangi globalin kullanılacağı belirtilir. Örnek: get, post, request gibi.

Http::input('get')->insert('zeroneed', 'ZN Framework');

echo Http::input('get')->select('zeroneed'); // Çıktı: ZN Framework

Http::input('get')->delete('zeroneed');

Config/Http.php Dosyasında Yer Alan Verileri Kullanmak

Http::code/message( string $type )

Config/Http.php dosyasında yer alan verileri kullanmak içindir.

ParametreAnlamları
1 - string TürHangi verinin kullanılacağı.

echo Http::code(404); // Çıktı: 404 Not Found
echo Http::message('notFound'); // Çıktı: 404 Not Found

Ajax Verisi Kontrolünü Yapmak

Http::isAjax()

Ajax işlemlerinde bulunulan isteğin ajax verisi olup olmadığını kontrol etmek için kullanılır.

if( ! Http::isAjax() )
{
	// redirect()
}

Views(Pages) Sayfa Tasarımı Dahil Etmek

Views/ dizini içerisindeki sayfaların dahil edilmesi veya çağrılması için kullanılır. Bu işlemi page() veya view() yöntemlerinden herhangi biri ile yapabilirsiniz.

ParametrelerAnlamları
string Dosya YoluÇağrılacak dosyasının yolu.
array Gönderilecek DataGönderilecek data.
[boolean Son Parametre = false]Çağrılanlar doğrudan çıktılasın mı? Yoksa bir değişkene mi aktarılsın? True olması durumunda bir değişkene aktarılarak kullanılabilir. Aksi halde doğrudan çıktı üretir.

Import::page("anasayfa", $veriler, false);

Ya da;

Import::view("anasayfa", $veriler, false);

Ya da;

Import::data($veriler)->view("anasayfa");

Yakaridaki kullanımda değişkene aktarılmak istenirse;

Import::data($veriler)->usable()->view("anasayfa");

Şablon Dosyası Dahil Etmek

Resources/Templates/ ve System/Templates dizinleri içerisindeki dosyaların yüklenmesi için kullanılır. Genellikle bu dizinlerde html içerikli tablolar, şemalar, formlar yer alır.

ParametrelerAnlamları
string Dosya YoluÇağrılacak dosyasının yolu.
array Gönderilecek DataGönderilecek data.
[boolean Son Parametre = false]Çağrılanlar doğrudan çıktılasın mı? Yoksa bir değişkene mi aktarılsın? True olması durumunda bir değişkene aktarılarak kullanılabilir. Aksi halde doğrudan çıktı üretir.

Import::template("EmailTemplate", $veriler);

Fonksiyon Dosyası Dahil Etmek

Application/Functions/Handload dizini içerisinde yer alan fonksiyon içerikli dosyalaları dahil etmek için kullanılır. Uzantı belirtmeye gerek yoktur.

ParametrelerAnlamları
string Dosya YoluÇağrılacak dosyasının yolu.

Import::handload("funcFile1", "funcFile2");

Script Dosyası Dahil Etmek

Resources/Scripts/ dizinindeki kendi oluşturduğumuz script dosyalarını dahil etmek için kullanılır. Parametre olarak dahil edeceğimiz script dosyalarının isimlerini kullanırız. Uzantılarını yazmaya gerek yoktur. Aynı anda birden fazla dosyayı yüklemeniz mümkündür.

ParametrelerAnlamları
string Dosya YoluÇağrılacak script dosyasının yolu.
[boolean Son Parametre = false]Çağrılanlar doğrudan çıktılasın mı? Yoksa bir değişkene mi aktarılsın? True olması durumunda bir değişkene aktarılarak kullanılabilir. Aksi halde doğrudan çıktı üretir.

Import::script("Validation");

Birden çok script dosyasını aynı anda dahil etmek istersek aşağıdaki şekilde kullanırız.

Import::script("Upload","Validation","Form");

Eğer import edilen script dosyasını bir değişkene veri olarak aktarmak istersek son parametre true olarak girilmelidir.

$script = Import::script("Upload","Validation","Form", true);

Ya da;

$script = Import::usable()->script("Upload","Validation","Form");

NOT
Config/Links.php dosyasında tanımlı Jquery ve JequeryUi sınıflarına ait URL bilgileri mevcuttur.

Jquery veya JqueryUi sınıflarını dahil etmek için aşağıdaki gibi kullanabilirsiniz.

Import::script("Jquery","JqueryUi");

Css Stil Dosyası Dahil Etmek

Resources/Styles/ dizini içerisine oluşturulmuş stil dosyalarını dahil etmek için kullanılır. Aynı anda birden fazla dosyayı dahil edebilmeniz mümkündür.

ParametrelerAnlamları
string Dosya YoluÇağrılacak stil dosyasının yolu.
[boolean Son Parametre = false]Çağrılanlar doğrudan çıktılasın mı? Yoksa bir değişkene mi aktarılsın? True olması durumunda bir değişkene aktarılarak kullanılabilir. Aksi halde doğrudan çıktı üretir.

Import::style("site");

Birden çok stil dosyasını aynı anda dahil etmek istersek aşağıdaki şekilde kullanırız.

Import::style("site","iletisim","panel");

Eğer import edilen stil dosyasını bir değişkene veri olarak aktarmak istersek son parametre true olarak girilmelidir.

$style = Import::style("site","iletisim","panel", true);

NOT
Dahil edeceğiniz stil dosyasının .css uzantısını yazmanıza gerek yoktur.

Font Dosyası Dahil Etmek

Resources/Fonts/ dizini içerisine eklenmiş font dosyalarını dahil etmek için kullanılır. Desteklenen fontlar SVG, WOFF, OTF, TTF, EOT uzantılı fontlardır.

ParametrelerAnlamları
string Dosya YoluÇağrılacak font dosyasının yolu.
[boolean Son Parametre = false]Çağrılanlar doğrudan çıktılasın mı? Yoksa bir değişkene mi aktarılsın? True olması durumunda bir değişkene aktarılarak kullanılabilir. Aksi halde doğrudan çıktı üretir.

Import::font("Elips");

Birden çok font dosyasını aynı anda dahil etmek istersek aşağıdaki şekilde kullanırız. Aynı anda birden fazla font dahil edilebilir.

Import::font("Elips","Abode","Secrcode/secrcode");

Eğer import edilen stil dosyasını bir değişkene veri olarak aktarmak istersek son parametre true olarak girilmelidir.

$font = Import::font("Elips","Abode","Secrcode/secrcode", true);

NOT
Fontlar bir dizinin içerisinde ise önce dizinin adı sonra font adı şeklinde yazılmalıdır. Ayrıca font uzantasını yazmanıza gerek yoktur.

ÖNEMLİ
Desteklenen font uzantıları. SVG, WOFF, EOT, OTF, TTF'dir

Herhangi Bir Dosya Dahil Etmek

.php, .css veya .js uzantılı dosyaları dahil etmek için kullanılır.

ParametrelerAnlamları
1. Parametre string Sayfa YoluÇağrılacak sayfanın yolu.
2. Parametre array Gönderilecek VeriÇağrılan sayfaya veri göndermek için kullanılır. Sadece PHP uzantılı sayfalar için bu parametre kullanılabilir.
[boolean 3. Parametre = false]Sayfanın doğrudan çıktılasın mı? Yoksa bir değişkene mi aktarılsın? True olması durumunda bir değişkene aktarılarak kullanılabilir. Aksi halde doğrudan çıktı üretir.

CSS uzantılı dosyaları dahil etmek

Import::something(STYLES_DIR.'site.css');

Bir değişkene aktarmak

$css = Import::something(STYLES_DIR.'site.css', '', true);

PHP uzantılı dosyaları dahil etmek

Import::something(CONTROLLERS_DIR.'iletisim.php');

PHP uzantılı dosyalara veri göndermek

Import::something(CONTROLLERS_DIR.'iletisim.php', array('data1' => 1, 'data2' => 2));

Bir değişkene aktarmak

$page = Import::something(CONTROLLERS_DIR.'iletisim.php', array('data1' => 1, 'data2' => 2), true);

JS uzantılı dosyaları dahil etmek

Import::something(SCRIPTS_DIR.'Jquery.js');

Bir değişkene aktarmak

$js = Import::something(SCRIPTS_DIR.'Jquery.js', '', true);

NOT
Böyle bir kullanımda dizin yollarını ve dosya uzantılarını yazmak zorundasınız.

Paket Dosyaları Dahil Etmek

Bu yöntemin temel işlevi bir dosya veya bir dizine ait tüm dosyaları tek seferde dahil etmek için kullanılır. 1. parametre 2 tür parametre içerir eğer tek bir dosya veya dizin dahil edilmek isteniyorsa dizge türü birden falza dosya veya dizin içi dosyaları dahil etmek isterseniz array türü belirtmeniz gerekmektedir.

ParametrelerAnlamları
1. Parametre string/array Sayfa YoluÇağrılacak sayfanın yolu.
[2. Parametre bool Recursive(İç içelik)]Belirtilen dizinin alt dizinlerinde arama. Varsayılan false.
[3. Parametre boolean Son Parametre = false]Çağrılanlar doğrudan çıktılasın mı? Yoksa bir değişkene mi aktarılsın? True olması durumunda bir değişkene aktarılarak kullanılabilir. Aksi halde doğrudan çıktı üretir.
[4. Parametre string Dizin]Bu parametre ile istenilen dizinin yolu belirtilebilir.

Herhangi bir dizin içindeki dizinlerin içerisinde yer alan .php, .css veya .js uzantılı dosyaların aynı anda dahil edilmesini sağlamak için kullanılır. Parametre olarak yazmanız gerekir.

Import::package("Slider");

Yukarıdaki kod Slider dizinindeki .php, .css ve .js uzantılı dosyaları otomatik olarak yükleme işlemi gerçekleştirir.

Dizin içi dizinler olduğu varsayılan yüklemeler yapmak için 2. parametre true olarak ayarlanmalıdır.

Import::package("Slider", true);

Bir değişkene aktarılarak kullanılmak istenirse 3. parametre true olarak ayarlanmalıdır.

Import::package("Slider", false, true);

İstenirse çok sayıda dosya ve dizinde 1. parametrenin array verü türü içermesi sureti ile dahil edilebilir. Aşağıda bir slider dizini içindeki dosyaların ve x.js dosyasının dahil edilmesi sağlanmıştır.

Import::package(array("Slider", "x.js"));

ÖNEMLİ
Böyle bir kullanımda dizin adı yazmak zorundasınız. O dizin içerisindeki dosyaları otomatik olarak yükleyecektir.

Tema Dosyaları Dahil Etmek

Resources/Themes/ dizini içindeki dosyaların yer aldığı tema dizininin adı belirtilerek dizin içindeki tema ile alakalı her dosyanın otomatik olarak yüklenmesi sağlanır.

Tema dizini içinde hangi uzantılı dosyalar yer alabilir? js, css, eot, svg, ttf, woff, php ve Config/Font.php dosyasında yer alan diğer font uzantıları.

ParametrelerAnlamları
1. Parametre string/array Tema AdıÇağrılacak temanın adı. Resources/Themes/ dizini için yer alan tema dizini adı. Birden fazla dosya veya dizin içi dosyalar yükletilecekse bu parametre array türde belirtilebilir.
[2. Parametre bool Recursive(İç içelik)]Belirtilen dizinin alt dizinlerinde arama. Varsayılan false.
[3. Parametre boolean Son Parametre = false]Çağrılanlar doğrudan çıktılasın mı? Yoksa bir değişkene mi aktarılsın? True olması durumunda bir değişkene aktarılarak kullanılabilir. Aksi halde doğrudan çıktı üretir.

Import::theme('Tema');

Birden fazla dosya veya dizinde dahil etme işlemi yapabilmek için 1. parametre array veri türünde belirtilir.

Import::theme(array('Tema', 'Tema2', 'style.css'));

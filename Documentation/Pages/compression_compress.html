Sıkıştırılmış Dosyayı Çıkarmak

Compress::extract( string $kaynak , string $hedef )

Sıkıştırılmış dosyayı çıkarmak için kullanılır.

ParametrelerAnlamları
Destekleyen Sürücülerrar, zip
1 - string KaynakÇıkarma işlemin yapılacağı sıkıştırılmış dosya.
2 - string HedefDosyaların hangi yola çıkarılacağı.

Compress::extract('test.zip', 'abc/')

Farklı Sürücüler Kullanmak

Compress::driver( string $surucu_ismi )->...

Farklı bir sürücü kullanmak için bu yönteme parametre olarak geçerli sıkıştırma sürücülerinden birinin ismi yazılır.

ParametrelerAnlamları
1 - string Sürücü İsmiKullanılabilir sürücüler.

echo Compress::driver('gz')->encode('nesne');
echo Compress::driver('bz')->encode('nesne');

Dosyaya Sıkıştırılmış Veri Yazmak

Compress::write( string $dosyaAdi, string $veri )

Dosyaya sıkıştırılmış veri yazmak için kullanılır.

ParametrelerAnlamları
Destekleyen Sürücülerbz, gz
1 - string Dosya AdıVerinin yazılacağı dosya. Şayet dosya yoksa oluşturularak yazılır.
2 - string VeriDosyaya yazılacak veri.

Compress::write('test.txt', 'Deneme')

Dosyadan Sıkıştırılmış Veriyi Okumak

Compress::read( string $dosyaAdi )

Sıkıştırılmış dosyayı okumak için kullanılır.

ParametrelerAnlamları
Destekleyen Sürücülerbz, gz
1 - string Dosya AdıOkunacak dosya adı.

echo Compress::read('test.txt')

Dizgeyi Sıkıştırmak

Compress::compress( string $veri, [ numeric $seviye = -1 ] numeric $tur = ZLIB_ENCODING_DEFLATE )

Metinsel ifadeyi sıkıştırmak için kullanılır.

ParametrelerAnlamları
Destekleyen Sürücülerbz, gz, lzf
1 - string VeriVerinin yazılacağı dosya. Şayet dosya yoksa oluşturularak yazılır.
2 - [ numeric Seviye = -1 ]Sıkıştırma seviyesi.
3 - [ numeric Kodlama Türü = ZLIB_ENCODING_DEFLATE ] Şifreleme türü.

echo Compress::compress('Sıkıştırılacak veri.')

Sıkıştırılmış Dizgeyi Açmak

Compress::uncompress( string $veri, [ numeric $uzunluk = 0 ] )

Sıkıştırılmış metni açar.

ParametrelerAnlamları
Destekleyen Sürücülerbz, gz, lzf
1 - string VeriVerinin yazılacağı dosya. Şayet dosya yoksa oluşturularak yazılır.
2 - [ numeric Uzunluk = 0 ]Kodu çözülecek verinin azami uzunluğu.

echo Compress::uncompress( Compress::compress('Sıkıştırılmış Veri') ) // Return: Sıkıştırılmış Veri

Dizgeyi Şifrelemek

Compress::encode( string $veri, [ numeric $seviye = -1 ] numeric $tur = FORCE_GZIP )

Metinsel ifadeyi sıkıştırmak için kullanılır.

ParametrelerAnlamları
Destekleyen Sürücülergz, zlib
1 - string VeriVerinin yazılacağı dosya. Şayet dosya yoksa oluşturularak yazılır.
2 - [ numeric Seviye = -1 ]Sıkıştırma seviyesi.
3 - [ numeric Kodlama Türü = FORCE_GZIP ] Şifreleme türü.

echo Compress::encode('Sıkıştırılacak veri.')

Sıkıştırılmış Dizgeyi Açmak

Compress::decode( string $veri, [ numeric $uzunluk = 0 ] )

Sıkıştırılmış metni açar.

ParametrelerAnlamları

1 - string VeriVerinin yazılacağı dosya. Şayet dosya yoksa oluşturularak yazılır.

Dizgeyi Deflate ile Şifrelemek

Compress::deflate( string $veri, [ numeric $seviye = -1 ] numeric $tur = ZLIB_ENCODING_RAW )

Metinsel ifadeyi sıkıştırmak için kullanılır.

ParametrelerAnlamları
Destekleyen Sürücülergz, zlib
1 - string VeriVerinin yazılacağı dosya. Şayet dosya yoksa oluşturularak yazılır.

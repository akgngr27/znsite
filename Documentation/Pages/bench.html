Benchmark(Performans Test) Kütüphanesi

Benchmark kütüphanesi, yazacağınız sistemin veya kodların performansını test etmek amacıyla oluşturulmuştur.

Benchmark(Performans Test) Kütüphanesi ve Yöntemleri
Benchmark Testini BaşlatmakBenchmark::start()
Benchmark Testini BitirmekBenchmark::end()
Test Esnasında Geçen Süreyi ÖğrenmekBenchmark::elapsedTime()
PHP Betiğinin Kullandığı Bellek Miktarını ÖğrenmekBenchmark::memoryUsage()
PHP Betiğine Ayrılan Maksimum Bellek Miktarını ÖğrenmekBenchmark::maxMemoryUsage()
Bir PHP Betiğinin Kullandığı Bellek Miktarını HesaplamakBenchmark::calculatedMemory()
Test Esnasında Kullanılan Dosyaları ÖğrenmekBenchmark::usedFiles()
Test Esnasında Kullanılan Dosya Sayısını HesaplamakBenchmark::usedFileCount()

Benchmark Testini Başlatmak

Benchmark::start( [ string $test_adi ] )

Benchmark testini başlatır. Tek parametresi vardır. Test adı parametresi.

NoParametreAnlamları
1[ string Test Adı ]Başlatılan testin ismi.

Benchmark::start('test');
     // kodlar

Benchmark Testini Bitirmek

Benchmark::end( [ string $test_adi ] )

Benchmark testini bitirir. Tek parametresi avrdır. Test adı parametresi

NoParametreAnlamları
1[ string Test Adı ]Bitirilecek testin ismi.

Benchmark::start('test');
     // kodlar 
Benchmark::end('test');

Benchmark::start('test1');
     // kodlar 
Benchmark::end('test1');

Test Test Esnasında Geçen Süreyi Öğrenmek

Benchmark::elapsedTime( [ string $test_adi ] , [ numeric $ondalik = 4 ] )

Benchmark testi sonunda geçen süreyi öğrenmek için kullanılır.

NoParametreAnlamları
1[ string Test Adı ]Hangi testin geçen zamanı öğrenilmek isteniyorsa o testin adı yazılır.
2[ string/int Ondalık = 4 ]Elapsed Time yani geçen zaman süresinde virgülden sonraki kısmın kaç basamaklı olacağı belirlenir.

Benchmark::start('test');
     // kodlar 
Benchmark::end('test');

Benchmark::start('test1');
     // kodlar 
Benchmark::end('test1');

echo Benchmark::elapsedTime('test');  // 0.0895 

echo Benchmark::elapsedTime('test1');  // 0.0595

PHP Betiğinin Kullandığı Bellek Miktarını Öğrenmek

Benchmark::memoryUsage( [ boolean $gercek_bellek_miktari = false ] )

PHP kodlarının sistemde kaç byte yer kapladığını öğrenmek için kullanılır.

NoParametreAnlamları
1[ boolean Gerçek Bellek Miktarı = false ]Gerçek bellek miktarı.

echo Benchmark::memoryUsage(); // 775.7 KB

PHP Betiğine Ayrılan Maksimum Bellek Miktarını Öğrenmek

Benchmark::maxMemoryUsage( [ boolean $gercek_bellek_miktari = false ] )

PHP kodlarına ayrılan maksimum bellek miktarını öğrenmek için kullanılır.

NoParametreAnlamları
1[ boolean Gerçek Bellek Miktarı = false ]Gerçek bellek miktarı.

echo Benchmark::maxMemoryUsage(); // 926.3 KB

Bir PHP Betiğinin Kullandığı Bellek Miktarını Hesaplamak

Benchmark::calculatedMemory( [ string $test_adi ] )

Bir PHP kodunun bellekte ne kadar yer kapladığını hesaplamak için kullanılır.

NoParametreAnlamları
1[ string Test Adı ]Hangi testin kodlarının ne kadar yer kapladığını öğrenmek için kullanılır.

Benchmark::start('test');
     // kodlar 
Benchmark::end('test');

Benchmark::start('test1');
     // kodlar 
Benchmark::end('test1');

echo Benchmark::calculatedMemory('test');  // 138896 

echo Benchmark::calculatedMemory('test1');  // 144531

Test Esnasında Kullanılan Dosyaları Öğrenmek

Benchmark::usedFiles( [ string $test_adi ] )

Testte kullanılan dosyaların listesini dizi türünde çıktılar.

NoParametreAnlamları
1[ string Test Adı ]Hangi test.

Benchmark::start('test');
     // kodlar 
Benchmark::end('test');

output( Benchmark::usedFiles('test1') );  // array

Test Esnasında Kullanılan Dosya Sayısını Hesaplamak

Benchmark::usedFileCount( [ string $test_adi ] )

Testte kullanılan dosyaların saysını çıktılar.

NoParametreAnlamları
1[ string Test Adı ]Hangi test.

Benchmark::start('test');
     // kodlar 
Benchmark::end('test');

echo Benchmark::usedFileCount('test1');  // 10

Metni Filtrelemek

Filter::word/data( string $metin , string/array $filtrelenecek_ifadeler , [ string $degisecek_kelime = '[badwords]' ] )

Metinsel ifadeyi filtrelemek için kullanılır.

NoParametrelerAnlamları
1string MetinFiltreleme yapılacak metin.
2string/array Filtrelenecek KelimelerFiltrelenecek kelime veya kelimeler.
3[ string Değişecek Kelime = "[badwords]" ]Filtrelenen kelimelerin yerini alacak yeni kelime. Varsayılan: [badwords]

$metin = "Kötü ve argo <script>alert(1);</script>kelime kullanmak yasaktır.";
echo Filter::word($metin , array('kötü', 'argo', '<script>', '</script>'), '[Kötü Kelime]');
/* Çıktı: [Kötü Kelime] ve [Kötü Kelime] [Kötü Kelime]alert(1);[Kötü Kelime]kelime kullanmak yasaktır. */

Filter::data() yöntemi word() yönteminin diğer adıdır. isterseniz word() yerine data()'da kullanabilirsiniz.

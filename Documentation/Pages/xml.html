XML Kütüphanesi

XML belgesi oluşturma ile ilgili işlemler.

XML Belgesinin Versiyonunu Ayarlamak

XML::version( $versiyon = '1.0' )

XML belgesinin versiyon bilgisini ayarlamak için kullanılır.

ParametrelerAnlamları
1 - string VersiyonXML Versinin versiyon. Varsayılan:1.0

XML::version('1.1');

XML Belgesinin Karakter Kodlamasını Ayarlamak

XML::encoding( $kodlama = 'UTF-8' )

XML belgesinin karakter kodlamasını ayarlamak için kullanılır .

ParametrelerAnlamları
1 - string KodlamaXML Versinin karakter kodlaması. Varsayılan:UTF-8

XML::version('1.1')->encoding('UTF-8');

XML Verisi Oluşturmak

XML::build( array $elemanlar, [ $versiyon ], [ $kodlama ] )

XML verisi oluşturmak için kullanılır.

ParametrelerAnlamları
1 - array VerilerXML verisi içinde yer alacak elemanlar
Verilerİşlev
nameElemanın adı
attrElemana eklenecek olan özellik değer çiftlerinin belirtileceği dizi.
contentElemanın içeriği.
childElemanın altında başka elemanlar olacaksa content yerine child olarak belirtilir.
[ 2 - string Versiyon ]Belirtilmediği takdirde 1.0 ön tanımlıdır.
[ 3 - string Kodlama ]Belirtilmediği takdirde UTF-8 ön tanımlıdır.

$data = XML::build(array
        (
            'name'    => 'media',
            'attr'    => array('id' => 1),
            'child'   => array
            (
                 array('name' => 'video', 'attr' => array('id' => 2), 'content' => 'Vidyo'),
                 array
                 (
                      'name'  => 'music', 
                      'attr'  => array('id' => 3),
                      'child' => array
                      (
                           array('name' => 'video', 'attr' => array('id' => 2), 'content' => 'Vidyo'),
                           array('name' => 'video', 'attr' => array('id' => 2), 'content' => 'Vidyo'),
                           array('name' => 'video', 'attr' => array('id' => 2), 'content' => 'Vidyo')
                      )
                 )
            )
        )); 
        
Kaynak Kod:
<?xml version="1.0" encoding="UTF-8"?>
<media id="1">
	<video id="2">Vidyo</video>
	<music id="3">
		<video id="2">Vidyo</video>
		<video id="2">Vidyo</video>
		<video id="2">Vidyo</video>
	</music>
</media>

XML Versini Dosyaya Kaydetmek

XML::save( $dosyaAdi )

XML dosyası oluşturmak için kullanılır.

ParametrelerAnlamları
1 - string Dosya AdıKaydedilecek XML dosyasının adı. .xml uzantısı kullanmaya gerek yoktur.
2 - string XML VerisiKaydedilecek XML verisi.

XML::save('test', $data);  // test.xml

XML Dosyasını Yüklemek

XML::load( $dosyaAdi )

Bir XML belgesini yüklemek için kullanılır.

ParametrelerAnlamları
1 - string Dosya AdıYüklenecek XML dosyasının adı.

XML::load('test'); // test.xml

XML Verisini Ayrıştırmak

XML::parse( $xmlVerisi, $tur = 'object' )

Bir XML belgesini ayrıştırmak için kullanılır. Nesne veya dizi veri türünde bir dönüşüm yapmak için kullanılır.

ParametrelerAnlamları
1 - string XML VerisiAyrıştırılacak XML verisi.
2 - string TürHangi veri türüne dönüştürüleceği. Varsayılan::object
Türlerİşlev
objectNesne veri türüne çevirir.
arrayDizi veri türüne çevirir.

XML::parse($data, 'array'); // Return array

XML Verisini Nesne Veri Türüne Çevirmek

XML::parseObject( $xmlVerisi )

Bir XML belgesini nesne veri türüne dönüştürmek için kullanılır.

ParametrelerAnlamları
1 - string XML VerisiAyrıştırılacak XML verisi.

XML::parseObject($data); // Return object

XML Verisini Dizi Veri Türüne Çevirmek

XML::parseArray( $xmlVerisi )

Bir XML belgesini dizi veri türüne dönüştürmek için kullanılır.

ParametrelerAnlamları
1 - string XML VerisiAyrıştırılacak XML verisi.

XML::parseArray($data); // Return array

XML Verisini Json Veri Türüne Çevirmek

XML::parseJson( $xmlVerisi )

Bir XML belgesini dizi veri türüne dönüştürmek için kullanılır.

ParametrelerAnlamları
1 - string XML VerisiAyrıştırılacak XML verisi.

XML::parseJson($data); // Return json
